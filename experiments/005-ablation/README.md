## 结论

### 正面发现

1. **固定参数与自动优化结果几乎一致**：收敛率、原型、收敛时间都相同，表明 ODE 动力学对函数族选择不敏感——默认参数已经足够好
2. 固定参数省去 10% 的优化开销（~14s），对大规模实验有实际价值

### 局限

1. 当前仅测试了两种变体，最关键的消融（跳过 Phase 2/3）未执行
2. 函数族选择不敏感可能是因为当前输入规模太小，在更大规模上差异可能显现
3. 需要代码级修改才能实现完整的消融矩阵
