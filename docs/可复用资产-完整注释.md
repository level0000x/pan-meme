## 第一部分：公理系统 ✅

**来源**：`泛模因理论.md` §1.2（P1-P8 完整陈述）+ `formal-concept-analysis-proof.md` §一（公理1-3 + 前提0-2）

### P1-P8 泛模因公理

> **状态**：✅ 全链共用，无需修改。

**P₁（信息本体论）**。信息 𝕀 是现实的基本构成维度，与物质-能量并列。信息不可还原为物质或能量，且具有独立因果效力。

**P₂（泛模因实在论）**。泛模因类 𝒫 定义为满足三个独立条件的实体集合：
$$\mathcal{P} = \{x \mid \text{replicable}(x) \land \text{evolvable}(x) \land \text{identity\_preserving}(x)\}$$

**P₃（结构现实主义）**。∀x，其同一性 id(x) 不由物质组成决定，而由关系模式 R_x ⊆ (x ∪ context(x))² 决定。

**P₄（认知-建模统一性）**。人类认知过程 C 与形式化建模过程 M 共享同构核心：C(h) ≅ M(h)。
> 📖 路径3中此公理未被直接引用，但为 FCA ↑↓ 的认知根基。

**P₅（可逆性认识论）**。模型的每一步变换必须有明确的逆操作。∀合法建模 Φ，∃Φ⁻¹ 使 Φ⁻¹∘Φ = id_Dom(Φ)。

**P₆（计算即模因）**。任何图灵完备系统 T 本身是一个泛模因实例：T ∈ 𝒫。
> 📖 路径3中此公理未被直接引用。

**P₇（演化普遍性）**。若系统 S 同时具备复制 R、变异 V、选择 S_e 三个算子，则 S 表现出演化行为：
$$S_{t+1} = S_e \circ V \circ R(S_t)$$
> ⚠️ 这是路径3的灵魂公理。注意它是**离散映射**，不是连续流。

**P₈（层级涌现）**。泛模因构成偏序集 (𝒫, ⪯)，其中 x ⪯ y 表示 x 是 y 的构成部分。存在向上因果 state(x) ↦ state(y) 和向下因果 constraint(y) ↦ state(x)。

**公理 2（状态二分公理）**。对任意 x ∈ 𝒰，P(x) 为真 ⇔ ∃y ∈ 𝒰, y ≠ x 使得 x 在结构上从属于 y。P(x) 真 → x 已归类；P(x) 假 → x 未归类。

## 第二部分：FCA ↑↓ 引擎 ✅

**来源**：`formal-concept-analysis-proof.md` 全文 + `information-structural-dynamics.md` §1.3

> **状态**：✅ 已完成严格证明。以下为可直接引用的已知结果。

### 输入-输出约定

**输入**：词列表 W = {w₁, ..., w_m}，每个 w 是字符集 Σ 上的字符串。

**自暴露字符集**（不需外部注入）：
$$C = \bigcup_{w \in W} \text{chars}(w)$$

**形式背景**：𝕂 = (C, W, I)，其中 (c, w) ∈ I ⇔ c ∈ chars(w)。

**论域**：𝒰 = C ∪ W，|𝒰| = |C| + |W| = n。

### 已知结果 R1-R5（直接可用）

**R1（关系网络 Ψ）**。Ψ = (W ∪ C, E)，其中 (c, w) ∈ E ⇔ c ∈ chars(w)。Ψ 是二分图，字和词之间只有 containment 边。

**R2（有限收敛）**。∀x ∈ 𝒰，↑↓ 交替序列在 N ≤ |𝒰| 步内收敛到不动点。
> ✅ 定理 4.1，`formal-concept-analysis-proof.md`。

**R3（概念格 𝔅(𝕂)）**。收敛不动点构成形式概念集合：
$$\mathfrak{B}(\mathbb{K}) = \{(A, B) \mid A \subseteq C, B \subseteq W, A^\uparrow = B, B^\downarrow = A\}$$
𝔅(𝕂) 在子概念-超概念序 ⪯ 下构成**完备格**。
> ✅ 定理 6.1，`formal-concept-analysis-proof.md`。

**R4（信息闭包 M(x)）**。∀x ∈ 𝒰，其完全信息闭包：
$$M(x) = \bigcup_{k=0}^{\infty} S_k^x$$
包含了从 x 出发通过 ↑↓ 交替在 𝒰 中可达的全部实体。

**R5（信息守恒）**。↑↓ 只在 Ψ 上行走——行走不改变 Ψ 的边集。信息不丢失。

## 第三部分：Ψ → 加权图 → 归一化拉普拉斯 ✅

### 字-字共现权

**定义**：对任意 c₁, c₂ ∈ C（字符集），字 c₁ 与 c₂ 的共现权重为：
$$w_C(c_1, c_2) = |\uparrow(c_1) \cap \uparrow(c_2)|$$
其中 ↑(c) = {w ∈ W | (c, w) ∈ E}——包含字 c 的所有词。

**公理强制**：P₃ 要求关系模式有定量强度。"共现一次"和"共现一百次"是不同的关系模式。|↑(c₁) ∩ ↑(c₂)| 是字 c₁ 和 c₂ 共享词境的词数——由 Ψ 唯一决定，不是选择的。

**加权字图**：G_C = (C, E_C, w_C)，其中 (c₁, c₂) ∈ E_C ⇔ w_C(c₁, c₂) > 0。

其中 ↓(w) = {c ∈ C | (c, w) ∈ E}——构成词 w 的所有字。

**加权词图**：G_W = (W, E_W, w_W)。

### 归一化拉普拉斯

**定义**：给定加权图 G = (V, E, w)，度矩阵 D 和邻接矩阵 A 为：
$$D_{ii} = \sum_j w_{ij}, \quad A_{ij} = w_{ij}$$
归一化拉普拉斯：
$$\mathcal{L} = I - D^{-1/2} A D^{-1/2}$$

**公理强制**：P₃ 说关系模式决定身份。ℒ 是关系模式的谱编码——特征值和特征向量完整捕获了图的连通性、聚类结构。这不是"选了谱图论"——ℒ 是"用线性代数表达关系模式结构"的最简自伴紧致表示。任何其他图表示要么丢失信息（如邻接矩阵忽略归一化），要么过度参数化（如完整邻接矩阵不区分全局缩放）。

**关键性质**（来自 Chung 1997）：
- ℒ 实对称半正定
- 特征值 0 = λ₀ ≤ λ₁ ≤ ... ≤ λ_{n-1} ≤ 2（归一化拉普拉斯特有上界 2）
- λ₀ 的重数 = 图的连通分量数 β₀
- Perron 特征向量 u₀ ∝ D^{1/2}𝟏（全正）

## 第四部分：概念格 → DG 基 → 约束谱聚类 ⚠️

**来源**：`information-structural-dynamics.md` §3

### DG 基

**定义**：𝔅(𝕂) 的 Duquenne-Guigues 基 ℬ_DG 是蕴涵规则的最小生成集。每条规则 A→B 表示"任何包含 A 中所有字符的词必然包含 B 中所有字符"。

> ⚠️ **截断问题**：当前限制 |A|≤2, |B|≤2。这是计算效率选择，非理论强制。去掉截断则为纯理论。在论文中需诚实标注。

**公理强制**：DG 基是概念格的逻辑编码。P₈ 要求偏序结构不可丢失——DG 基以蕴涵规则形式编码了概念格中的全部子概念-超概念关系。

### Must-Link 约束矩阵 Q

**定义**：对每对满足 A→B 的词 wᵢ, wⱼ（wᵢ 覆盖 A 中全部字符，wⱼ 覆盖 B 中全部字符），施加：
$$Q_{ij} = 1 \quad \text{（wᵢ 和 wⱼ 必须被分配到同一社区）}$$

**公理强制**：DG 基的蕴涵规则是概念格的逻辑闭包。若 wᵢ 的结构从属于 wⱼ，则它们不能被划分到不同社区——那会切断 P₈ 的向上/向下因果链。

### 约束广义特征值问题

**定义**：
$$\mathcal{L} v = \lambda D v, \quad \text{s.t. } v^\top Q v = 0$$
其中 ℒ 是 G_W 的归一化拉普拉斯。约束 v^⊤Qv = 0 强制：若 Q_{ij}=1，则 v_i = v_j——被 must-link 绑定的词对在谱嵌入中重合。

> ⚠️ **需要加强的论证**：为什么约束形式必须是 v^⊤Qv = 0？需要证明在谱嵌入+图切割框架内，这是最小信息损失的 must-link 约束编码。

**公理强制**：P₃ 要求按关系模式分组（谱嵌入），P₈ 要求分组不可违反概念格层级（must-link）。两个要求同时施加的唯一解就是约束广义特征值问题。

### 社区划分

**定理**：约束广义特征值问题的前 k* 个特征向量 {v₁, …, v_k*} 给出 k* 维谱嵌入。在此嵌入上执行 k-means 聚类，获得社区划分 {X₁, …, X_k*}。

**社区数**：k* = min(|𝔅(𝕂)|, n − 1)——社区数等于概念数。每个形式概念是一个候选社区核心。
> ⚠️ k* = |𝔅(𝕂)| 这一公式与实验 010 的 649 个社区之间的数量关系需要验证。

**公理强制**：
- P₃：关系模式决定身份——形式概念 (A, B) 定义了一组共享关系模式的实体，它们是同一性候选
- P₈：层次不可被跨切——must-link 约束强制执行
- k-means 是给定谱嵌入和 k* 后唯一不需要额外参数的划分方法

## 第六部分：A/(A+B) 引擎 ✅ —— 全链核心资产

**来源**：`information-structural-dynamics.md` §5.1 引理 5.1

> **状态**：✅ 完成了论证核心——五条条件对应五条公理，唯一解是 A/(A+B)。**全项目最宝贵的证明。**
> **依赖**：无——这是纯函数形式定理

### 引理 5.1（增长/衰减竞争形式的唯一性）

**问题**：设一个维度的演化由增长力 A（什么让它增强）和衰减力 B（什么让它减弱）的竞争决定。要求：

| 条件 | 含义 | 强制公理 |
|------|------|----------|
| 1. 有界性 | 值域始终在 [0,1] | P₃：关系模式的强度有上限 |
| 2. 单调性 | ∂f/∂A ≥ 0, ∂f/∂B ≤ 0 | P₇：增长力是复制+变异，更强的力→更大的状态值 |
| 3. 标度不变性 | f(kA, kB) = f(A, B) | P₁：信息不可约化为"量"；只有结构比例有意义 |
| 4. 对称性 | f(A, B) + f(B, A) = 1 | P₈：双向因果是对称竞争的两面 |
| 5. 最简形式 | f(t, 1) 具最少独立参数 | P₅：可逆性要求模型拥有最简结构 |

**结论**：唯一满足全部五条件的最简解为：
$$\boxed{f(A, B) = \frac{A}{A + B}}$$

**证明**（简述）：条件 1→4 迫使 f(A, B) 是比值 A/(A+B) 的单调函数：取 f(A, B) = g(A/B)，由 f(1,1) = ½（条件 4 自反）且 g'(r) ≥ 0（条件 2）。条件 3 进一步迫使 g 是一维函数。条件 5 在 g(r^p) 族中选择 p=1。任何其他函数要么违反某条件，要么依赖额外自由参数。

**意义**：A/(A+B) 不是"选了一个好看的函数"——它是 P₁, P₃, P₅, P₇, P₈ 五条公理同时施加时**唯一剩下的形式**。

### 力的分配原则（P₈ 双向因果）

**向上因果（子→父）**：
- B_up（子社区的关联度）→ 父社区有更多"接口" → 推动父社区的 B ↑
- ρ_up（子社区的能流密度）→ 信息流向上汇聚 → 推动父社区的 ρ ↑

**向下因果（父→子）**：
- D（父社区的内禀度）→ 约束子社区的过度扩张 → 限制子社区的 B
- S（父社区的结构韧度）→ 为子社区提供结构锚定 → 稳定子社区的 R

### N 算子当前形式

$$\begin{aligned}
N_D(M) &= \frac{\alpha_1 R + \varepsilon_1}{\alpha_1 R + \beta_1 (B + B_{\text{up}}) + \varepsilon_1} \\[4pt]
N_B(M) &= \frac{\gamma_1 (R + B_{\text{up}}) + \varepsilon_2}{\gamma_1 (R + B_{\text{up}}) + \delta_1 D + \varepsilon_2} \\[4pt]
N_\rho(M) &= \frac{\zeta_1 (D + \rho_{\text{up}}) + \varepsilon_3}{\zeta_1 (D + \rho_{\text{up}}) + \eta_1 R + \varepsilon_3} \\[4pt]
N_R(M) &= \frac{\theta_1 (\rho + \rho_{\text{up}} + B_{\text{up}}) + \varepsilon_4}{\theta_1 (\rho + \rho_{\text{up}} + B_{\text{up}}) + \kappa_1 D + \kappa_2 S + \varepsilon_4} \\[4pt]
N_S(M) &= \frac{\lambda_1 D + \varepsilon_5}{\lambda_1 D + \mu_1 R + \varepsilon_5}
\end{aligned}$$

> ⚠️ **硬阻塞**：当前 N 算子有 11 个命名参数 + 5 个 ε = **16 个参数**。但谱参数只有 11 个。数量不对齐。
>
> **修复方案**：
> 1. 减少 N 算子参数——重新设计 N 使参数更少（如论证某些参数可以合并或由 A/(A+B) 的对称性消去）
> 2. 论证某些 N 参数可由谱参数的代数组合重复使用（如 κ₁=α₁, κ₂=β₂）
> 3. 检查是否 5 个 ε 可以统一为一个 ε（数值稳定项不增加信息维度）

### N 迭代

**定义**：M^{(k+1)} = N(M^{(k)})，M^{(0)} 为各社区的初始谱描述子。

**公理强制**：P₇——S_{t+1} = S_e ∘ V ∘ R(S_t) 是离散映射。每次 N 迭代对应一次完整 P₇ 周期：
- 复制：社区保持自身结构，传播到相邻社区（B_up 传导）
- 变异：谱结构根据耦合关系微调（R 维度驱动变化）
- 选择：A/(A+B) 竞争——最稳定的增长/衰减比生存

**不是 ODE 的离散化。** 离散迭代是第一性的——信息不以 dt→0 演化。ODE 可以是极限下的连续近似。

## 第八部分：不动点与收敛 🔲

**来源**：`information-structural-dynamics.md` §6

> **状态**：🔲 定理陈述正确，但压缩性目前仅有数值佐证（350+ 样本点）。解析证明是 P0 优先级待补项。
> **依赖**：N 算子（§7）

### 定理 6.1（N 在不动点附近的压缩性）

N 是 A/(A+B) 形式的复合。单变量 A/(A+B) 的 Jacobian 元素：
$$\frac{\partial}{\partial A}\frac{A}{A+B} = \frac{B}{(A+B)^2} \ge 0, \quad \frac{\partial}{\partial B}\frac{A}{A+B} = -\frac{A}{(A+B)^2} \le 0$$

这保证了 ∂N_i/∂M_j 的符号模式。∥J_N∥₁ < 1 的全局条件可以化简为参数空间上的不等式约束。

> ⚠️ **当前状态**：J_N(M*) 的特征值在实验 008/009/010 的全部 350+ 样本点上全部落在单位圆内。但这不是解析证明。"350 个例子"不构成"所有正参数下成立"的证明。

### 定理 6.2（不动点存在唯一性）

N 是 [0,1]⁵ → (0,1)⁵ 的连续映射（ε 项保证严格正）。Brouwer 不动点定理 → 不动点 M* = N(M*) 存在。在非退化参数下（J_N(M*) 满秩），不动点唯一。

> ✅ Brouwer 部分是严谨的。唯一性依赖 J_N 满秩假设——需要精确定义"非退化"。

### 推论 6.3（指数收敛）

若 N 在 M* 附近是压缩映射，存在 ρ ∈ (0,1) 使：
$$\|M^{(k)} - M^*\| \le \rho^k \|M^{(0)} - M^*\|$$
典型参数下 20–60 步内 ∥M^(K) − M*∥ < 10⁻⁶。

> 🔲 悬在定理 6.1 的解析证明上。

### P₅ 在动力学层的体现

P₅（可逆性认识论）→ N 迭代可追踪：N 是确定性映射，M^{(k)} 由 M^{(k-1)} 唯一决定。轨迹每一步可回溯。在不动点邻域内 N 局部可逆（J_N 满秩 → 隐函数定理）。

## 第九部分：信息作用量谱 ✅

**来源**：`information-structural-dynamics.md` §7

> **状态**：✅ 路径3最独特的创新输出。无需阈值。五个连续值的谱签名。
> **依赖**：N 迭代轨迹、不动点 M*

### 信息耗散泛函

$$\Phi(M) = n_i \cdot D \cdot (1 - S)$$
衡量社区偏离信息饱和状态的程度。D 越高、S 越低 → 耗散越大。

### 五分量谱签名

$$\begin{aligned}
S &= \sum_{k=0}^{K-1} \Phi(M^{(k)}) \cdot \Delta t \quad \text{——信息作用量（总耗散）} \\[4pt]
\Phi^* &= \Phi(M^*) = n_i \cdot D^* \cdot (1 - S^*) \quad \text{——基态耗散（不动点残余）} \\[4pt]
W_{\text{diss}} &= \sum_{k=0}^{K-1} \|\Phi(M^{(k+1)}) - \Phi(M^{(k)})\| \quad \text{——耗散功（总变化量）} \\[4pt]
\eta_{\text{info}} &= \frac{\|M^* - M^{(0)}\|}{(W_{\text{diss}})^{1/2}} \quad \text{——信息路径效率（直进度）} \\[4pt]
\tau^{-1} &= -\log \rho \quad \text{——弛豫速率（收敛速率的时间常数）}
\end{aligned}$$

其中 Δt = 1（每次 N 迭代为一个单位时间步）。

### 对应 P₂ 三条件

| 作用量分量 | 对应 P₂ 条件 | 含义 |
|-----------|-------------|------|
| S, W_diss | **可演化** | 经历多少变异-选择循环。S 大→高耗散演化，S 小→低耗散直进 |
| Φ* | **身份保持** | 不动点仍有残余耗散。完全信息饱和不可达→P₂ 三条件间存在内禀张力 |
| η_info | **可复制** | 路径是否高效。高 η→接近直线（高效自洽，易于复制） |
| τ⁻¹ | **三者共享** | 收敛多快。快的社区适应性强，慢的对初始条件敏感 |

### 优势

相比原型分类（链 A 的九类/链 B 的三族九子型），作用量谱的优势：
- 无需硬阈值——五个连续值天然可比较
- 避免了"这个社区是基石还是过客"的分类争议
- 每个社区的谱签名是其全程努力（而非终点状态）的忠实记录

### 信息守恒（定理 7.1）

N 迭代全程信息可追踪：M^{(k)} 由 M^{(k-1)} 和 N 唯一决定。N 是函数，非关系。轨迹每一步可回溯。局部逆存在（J_N 满秩 → 隐函数定理保证邻域内可逆）。

## 第十一部分：热迹参数映射框架 📖

**来源**：`pan-meme-complete-proof.md` §5-7 + `spectral-bridge-design.md` 补全 4-5

> **状态**：📖 参考用。用于修复路径3当前"11 个谱参数到 N 算子参数"的闭式映射缺口。
> **使用方式**：不是直接复制，而是借鉴其"从统一数学对象衍生的"参数构造逻辑。

### 热迹渐近展开 —— 四标度点的数学必然性

归一化拉普拉斯热迹 Θ(t) = Σ e^{-tλ_k} 在谱间隙存在时，有四个天然标度点：

| 标度点 | 渐近区域 | 数学特征 |
|--------|----------|----------|
| t*/2 | 混合前中期 | 同时依赖小 t 展开和大 t 间隙——信息量最大 |
| t* | 临界混合 | Θ(t*) = β₀ + 1——精确的混合阈值 |
| 2t* | 衰减段 | Θ(2t*) − β₀ ≈ e^{-2t*λ₁}——间隙的二阶信息 |
| 3t* | 长尾 | Θ(3t*) − β₀ ≈ e^{-3t*λ₁}——间隙的高阶验证 |

**关键**：{t*/2, t*, 2t*, 3t*} 不是任意选的——少于这四个点会丢失至少一个渐近阶的信息。β₀+1 而非 β₀+0.5 的理由：1 是结构变化的原子单位（新增一个孤立顶点 = β₀ → β₀+1）。

### 11 参数谱映射公式（链 B 补全 5）

$$\boxed{
\begin{aligned}
\alpha_1 &= \frac{\lambda_1}{2|E|}, \quad \alpha_2 = \frac{\Theta_{0.5}}{n}, \quad \beta_1 = \frac{\lambda_{\max} - \lambda_1}{\lambda_{\max}}, \quad \beta_2 = \frac{\sum \lambda_k^2}{(\sum \lambda_k)^2}, \\[4pt]
\gamma_1 &= 1 - \frac{\Theta_2}{\Theta_1}, \quad \gamma_2 = \frac{\Theta_{0.5}}{\Theta_1} - 1, \quad \delta_1 = \frac{\Theta_{0.5} - \Theta_1}{n}, \\[4pt]
\delta_2 &= \frac{\lambda_{\max}}{\sum_{k > 0} e^{-\lambda_k t^*}}, \quad \delta_3 = 1 - \frac{\Theta_3}{\Theta_2}, \quad \varepsilon_1 = \frac{\lambda_1}{\lambda_{\max}}, \quad \varepsilon_2 = 1 - \frac{\Theta_2}{n}
\end{aligned}
}$$

**源**：所有参数从 Θ(t) 在四个标度点的取值闭式导出。不依赖任何经验系数或手工映射。

**值域**：全部 ∈ [0,1]（或 (0,1]），由 Θ(t) 的单调性和归一化拉普拉斯的 λ_k ∈ [0,2] 保证。

**量纲**：归一化拉普拉斯从根上消去了边权物理量纲——λ_k ∈ [0,2] 是无量纲比值。所有含 λ_k 的参数均为无量纲量的代数组合。

**维度**：11 个参数，独立谱量只有 6 个（{n, β₀, λ₁, λ_max, Θ_{0.5}, Θ_2}），5 个是必要代数组合。

### 如何使用这个框架修复路径3的参数映射

路径3 当前 N 算子的 11 个命名参数（α₁, β₁, γ₁, δ₁, ζ₁, η₁, θ₁, κ₁, κ₂, λ₁, μ₁）需要用"从谱/热迹数据导出的闭式"填充。链 B 提供了如何从 Θ(t) 导出 11 个参数的完整范例。但路径3 的 N 算子和链 B 的 ODE 参数含义不同——不能直接复制数值，需要建立对应关系或重新设计 N 算子使其参数减少到与独立谱参数数量匹配。
