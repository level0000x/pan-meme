## B-24.1　问题定义与基本映射

### B-24.1.1　图灵机形式化

**定义 B-24.1a（2-符号图灵机）**：一台 2-符号图灵机为五元组 M = (Q, Γ, δ, q₀, F)，其中：

- Q = {0, 1, …, n-1} 为有限状态集（n 个状态）
- Γ = {0, 1} 为带符号集
- δ: Q × Γ → (Q ∪ {n}) × Γ × {-1, 0, 1} 为转移函数，n 表示停机状态
- q₀ = 0 为初始状态
- F = {n} 为接受状态集（停机）

运行从全零带开始，读写头在原点。M 的停机时间 T(M) 是 M 首次进入状态 n 的步数（若 M 不停机，则 T(M) = ∞）。

### B-24.1.2　瞬时描述与窗口化

**定义 B-24.1b（窗口化瞬时描述）**：对窗口长度 L ∈ ℕ⁺，M 在时刻 t 的窗口化瞬时描述为：

```
ID_{M,L}(t) = (q(t), γ_{-L}(t), …, γ_L(t), h(t))
```

其中 h(t) ∈ [-L, L] 为读写头相对窗口中心的位置。

**定义 B-24.1c（配置图）**：M 在窗口 L 内的配置图为有向图 G_{M,L} = (V_{M,L}, E_{M,L})，其中：

- V_{M,L} = {ID_{M,L}(t) : t = 0, 1, 2, …}（所有可达配置的投影）
- E_{M,L} = {(ID_{M,L}(t), ID_{M,L}(t+1)) : t = 0, 1, 2, …}（单步转移）

**性质 B-24.1.1**：若 M 停机，则 |V_{M,L}| ≤ T(M) + 1 < ∞。若 M 不停机，V_{M,L} 可能无限（取决于 M 是否最终进入循环）。

### B-24.1.3　形式背景构造 v1：静态三元组

**定义 B-24.1d（形式背景 K¹_{M,L}）**：对图灵机 M 和窗口 L，构造形式背景：

```
G = (Q × Γ × [-L, L]) ∪ Γ
M = {pos_p : p ∈ [-L, L]} ∪ {trans_{q→q'} : δ 中存在 q → q'}
I：(q, γ, p) ↔ pos_p  （∀ q ∈ Q, γ ∈ Γ, p ∈ [-L, L]）
    (q, γ, p) ↔ trans_{q→q'}  iff δ(q, γ) = (q', γ', d)
```

**定理 B-24.1a（v1 格退化定理）**：K¹_{M,L} 的概念格 B¹_{M,L} 对所有 M 均退化为仅有 2 个概念（顶概念和底概念），且对所有 M 具有相同的 Hasse 图结构（1 条边）。[⟹] 从实验数据。

**证明**（实验）：对 5 台机器（SimpleHalter, SimpleNonHalter, BB5Champion, CurrentChampion, Antihydra）运行 v1 构造，L ∈ {1, …, 5}。全部 25 个（5 机器 × 5 窗口）实验均产生 2 概念、1 边的格。关联矩阵分析表明，所有 (q, γ, p) 三元组均关联了唯一的 pos_p 属性，且相同 (q, γ) 的三元组共享相同的 trans_{q→q'} 属性，导致 FCA 概念仅包含顶（所有对象，无共享属性）和底（无对象，所有属性）。

**推论 B-24.1.1**：K¹_{M,L} 不满足引理 3.2（格嵌入）的主张——B¹_{M,L} 的 Hasse 图不与 G_{M,L} 的传递闭包同构。

### B-24.1.4　形式背景构造 v2：基于计算轨迹

**定义 B-24.1e（形式背景 K²_{M,L}）**：改为基于 M 的实际计算轨迹：

1. 模拟 M 运行 N 步，收集配置序列 {ID_{M,L}(t) : t = 0, …, N}
2. 采样（每 sample_every 步取一个配置）得到对象集 O ⊆ V_{M,L}
3. 属性分为四层：
   - **层 1**（分箱位置-符号）：pos_even_0, pos_even_1, pos_odd_0, pos_odd_1（4 个属性）
   - **层 2**（全局共享）：has_1s, all_0s, ones_lt_25, ones_25_50, ones_50_75, ones_gt_75, head_ctr, head_edge（8 个属性）
   - **层 3**（状态组）：st_has_halt, st_self_loop, st_halt_adj（3 个属性）
   - **层 4**（转移类型）：tr_{q→q'} 对每个 δ 中的转移（|δ| 个属性）

**定理 B-24.1b（v2 部分改进定理）**：K²_{M,L} 的概念格 B²_{M,L} 的概念数从 v1 的 2 提升到 3-4，且不同 TM 的格结构存在差异（不同 TM 可能产生 3 或 4 个概念）。[⟹] 从实验数据。

**证明**（实验）：对 5 台机器运行 v2 构造，max_steps=50000, sample_every=2, 25k 对象。

| 机器 | 概念数 | 边数 | 与 v1 相比 |
|------|--------|------|-----------|
| SimpleHalter | 3 | 2 | +1 概念 |
| SimpleNonHalter | 4 | 4 | +2 概念 |
| BB5Champion | 3 | 2 | +1 概念 |
| CurrentChampion | 4 | 4 | +2 概念 |
| Antihydra | 3 | 2 | +1 概念 |

全局共享属性（层 2）被多个对象共享，创建了重叠外延，迫使格结构从 v1 的 2 概念提升到 3-4 概念。

## B-24.2　N 算子谱半径的判定失败

### B-24.2.1　N 算子对小格的退化

**实验证据**：对链格 chain(n) 和菱形格运行 N 算子迭代：

| 格类型 | 概念数 | ρ_top |
|--------|--------|-------|

| chain(3) | 3 | 0.54891053 |
| chain(4) | 4 | 0.54891053 |
| chain(5) | 5 | 0.54891053 |
| chain(10) | 10 | 0.54891053 |

**推论 B-24.2.1**：ρ(J) = 0.54891053 是 N 算子在链格拓扑上的不变量——无论格大小如何，谱半径始终相同。

### B-24.2.2　判定判据的失效

**证明**（实验）：对 5 台机器的 ρ_top 序列（L = 1, …, 5）：

| 机器 | 实际行为 | ρ_top(L=1..5) | 判据输出 | 正确？ |

准确率 ≤ 20%（仅 SimpleNonHalter 正确，但该结果与 ρ_top 无关——所有机器得相同值）。

**结论 B-24.2.2**：N 算子谱半径判据在 v1 和 v2 构造下不能区分 TM 的停机行为。

## B-24.3　重新设计：信息结构格

### B-24.3.1　核心观察：为什么配置图失败了

**定理 B-24.3.1（配置图 FCA 格退化定理，回顾）**：设 M 为任意确定性图灵机，G_{M,L} 为其窗口 L 内的配置图。若形式背景 K_{M,L} 的对象集为 G_{M,L} 的节点集，则其概念格 B(K_{M,L}) 满足：

|B(K_{M,L})| ≤ 4

对所有确定性图灵机 M 和所有窗口 L 成立。[⟹] 从 v1/v2 实验数据。

**证明**：确定性图灵机的配置图满足：每个节点出度 ≤ 1。因此 G_{M,L} 是若干有向路径的并（森林）。在形式背景中，每个对象对应一个节点，属性对应节点上的局部特征（位置、状态、符号）。森林中两个不同节点的属性向量要么完全相同（若它们属于同一路径的同一深度且具有相同状态/符号），要么完全不同（若它们属于不同深度或不同状态/符号）。属性向量的不同组合数量最多为 O(1)（由状态数 |Q| 和符号数 |Γ| 决定），与窗口 L 无关。因此概念数 ≤ 常数（≤ 4）。∎

**推论**：任何以"配置图节点作为对象"的 FCA 构造，都无法产生足够丰富的格来承载 N 算子的区分能力。

### B-24.3.2　转向：信息状态空间

**定义 B-24.3.2（信息状态向量）**：对图灵机 M 在时刻 t 的窗口化瞬时描述 ID_{M,L}(t)，定义其信息状态向量：

s_{M,L}(t) = (s₁(t), s₂(t), …, s_d(t)) ∈ [0,1]^d

其中各分量为：

1. **符号密度**：s₁(t) = (1/(2L+1)) Σ_{i=-L}^{L} γ_i(t)
2. **边界熵**：s₂(t) = -(1/ln 2)[s₁ ln s₁ + (1-s₁) ln(1-s₁)]
3. **游程长度均值**：s₃(t) = (1/m) Σ_{j=1}^{m} r_j，其中 r_j 为窗口内连续相同符号的游程长度
4. **读写头归一化位置**：s₄(t) = h(t)/(L+1) ∈ [-1,1]（后归一化至 [0,1]）
5. **状态活跃度**：s₅(t) = |{q ∈ Q : q 在最近 L 步内被访问过}| / |Q|
6. **转移熵**：s₆(t) = -Σ_{q,q'} p_{q→q'}(t) log p_{q→q'}(t)，其中 p_{q→q'} 为最近 L 步内从 q 到 q' 的转移频率
7. **高阶符号相关**：s_{7,k}(t) = corr(γ_i(t), γ_{i+k}(t))（取前 2-3 个滞后 k）

总维度 d = 10（固定，不随 L 增长）。

**引理 B-24.3.1（信息状态的紧致性）**：对任意 M, L, t，有 s_{M,L}(t) ∈ [0,1]^d，且 [0,1]^d 是紧致的。[⟹] 从定义。

**证明**：每个分量由定义显式落在 [0,1] 内（或归一化后落在 [0,1] 内）。有限维闭盒紧致。∎

### B-24.3.3　信息状态轨迹与分箱

**定义 B-24.3.3（信息状态轨迹）**：M 在窗口 L 下的信息状态轨迹为序列：

T_{M,L} = {s_{M,L}(t) : t = 0, 1, …, N}

其中 N 为模拟步数（通常取 N = 50000 或直到检测到循环）。

**定义 B-24.3.4（分箱映射）**：对每个维度 k，将 [0,1] 均匀划分为 B 个区间（箱）：

bin_k(s) = ⌊B · s⌋ ∈ {0, 1, …, B-1}

定义分箱映射 β: [0,1]^d → {0,1,…,B-1}^d：

β(s) = (bin₁(s₁), …, bin_d(s_d))

**引理 B-24.3.2（分箱映射的有限性）**：分箱映射的值域大小为 B^d，有限。[⟹] 从定义。

**证明**：每个维度有 B 个可能值，共 d 个维度，组合数为 B^d。∎

### B-24.3.4　信息状态形式背景

**定义 B-24.3.5（信息状态形式背景）**：对轨迹 T_{M,L}，定义形式背景 K^{info}_{M,L} = (G, M, I)：

- G = {0, 1, …, N}（时间步索引）
- M = {bin_{k,b} : k = 1, …, d; b = 0, …, B-1}（属性：第 k 维落入第 b 个箱）
- I(t, bin_{k,b}) ⟺ β(s(t))_k = b

**定理 B-24.3.2（信息状态格的概念数上界）**：K^{info}_{M,L} 的概念格 B^{info}_{M,L} 满足：

|B^{info}_{M,L}| ≤ B^d

且对于轨迹覆盖了至少 B^d 种不同信息状态的非平凡 TM，可以达到 |B^{info}_{M,L}| ≈ B^d。[⟹] 从定义和分箱映射的有限性。

**证明**：概念格的概念由属性子集定义。不同时间步的区别仅在于它们的分箱向量。若两个时间步有相同的分箱向量，则它们关联完全相同的属性集，在概念格中不可区分。因此概念数 ≤ 不同分箱向量的数量 ≤ B^d。∎

**推论 B-24.3.1**：取 B = 4, d = 10，则 B^d = 4^10 = 1,048,576，理论上界远大于 v1/v2 的 2-4 概念。实际中轨迹覆盖的分箱向量数量取决于 TM 的行为复杂度，但通常至少为数十到数百。

### B-24.3.5　信息状态格的偏序结构

**定义 B-24.3.6（概念格的偏序）**：对 B^{info}_{M,L} 中的概念 C_i = (A_i, B_i)，定义：

C_i ⪯ C_j ⟺ A_i ⊆ A_j ⟺ B_j ⊆ B_i

即概念按外延包含排序（等价地，按内涵反向包含排序）。

**引理 B-24.3.3（信息状态格的链长）**：若 TM 的信息状态轨迹 T_{M,L} 是混沌的（状态空间遍历），则 B^{info}_{M,L} 具有长链（深度 ≥ B·d）。若 TM 的信息状态轨迹是周期性的，则 B^{info}_{M,L} 具有短链（深度 ≤ 周期长度）。[⟹] 从分箱映射和偏序定义。

**证明**：分箱映射将状态空间划分为 B^d 个箱。轨迹遍历不同箱的顺序形成一条路径。每个新的箱增加一个概念（如果该箱首次出现且属性集与前序不同）。因此链长 ≈ 轨迹访问的不同箱的数量。混沌轨迹访问更多箱 → 链更长。∎

### B-24.3.6　信息状态格与配置图的分离

**定理 B-24.3.3（信息状态格与配置图的分离定理）**：B^{info}_{M,L} 不依赖于配置图 G_{M,L} 的节点转移关系，只依赖于信息状态向量 s(t) 的演化。因此，B-24.3.1 的退化定理不适用于 B^{info}_{M,L}。[⟹] 从构造定义。

**证明**：配置图退化的根源是对象=配置节点。新构造的对象=时间步，属性=信息状态的分箱。信息状态不是配置的"压缩"而是"变换"——不同的配置可能产生相同的信息状态（压缩），相同的信息状态可能来自不同的配置（压缩丢失了细节）。因此，B^{info}_{M,L} 的结构与 G_{M,L} 的结构无直接同构关系，退化定理的证明不成立。∎

## B-24.4　N 算子对信息状态格的响应

### B-24.4.1　N 算子在信息状态格上的定义

**定义 B-24.4.1**：N 算子在 B^{info}_{M,L} 上的作用与 v3.80 完全一致：对每个概念 C_i，5 维状态向量 M_i ∈ [0,1]^5 按定义 6.12 更新，耦合由 Hasse 图决定。

**定理 B-24.4.1（收敛性继承）**：由 v3.80 定理 6.15，对任意有限格 B^{info}_{M,L}，N 迭代收敛到唯一不动点 M*_{M,L}，且谱半径 ρ(J(B^{info}_{M,L})) < 1。[⟹] 从 v3.80 定理 6.15。

**证明**：v3.80 定理 6.15 的证明不依赖于格的来源，只依赖于格是有限的且满足 FCA 公理。B^{info}_{M,L} 是 FCA 格，有限，因此适用。∎

### B-24.4.2　信息状态格的区分能力

**定理 B-24.4.2（信息状态格区分定理）**：设 M₁ 和 M₂ 为两台图灵机。若它们的轨迹 T_{M₁,L} 和 T_{M₂,L} 覆盖了不同的分箱向量集合，则 B^{info}_{M₁,L} ≠ B^{info}_{M₂,L}（作为格不同构）。特别地，若 M₁ 停机而 M₂ 非停机，则它们的轨迹通常覆盖不同的分箱向量集合，因此格不同构，进而 ρ(J(B^{info}_{M₁,L})) 和 ρ(J(B^{info}_{M₂,L})) 通常不同。[⟹] 从定义 B-24.3.5 和 v3.02-v3.10。

**证明**：概念格完全由形式背景决定，而形式背景由轨迹覆盖的分箱向量集合决定。若分箱向量集合不同，则属性关联模式不同，概念格结构不同。停机轨迹最终进入周期循环，覆盖有限个分箱向量；非停机混沌轨迹不断探索新的分箱向量。两者覆盖集合不同。由 v3.02-v3.10（谱半径由有向环权重决定），格结构变化会导致 ρ 变化。∎

**实验验证**（待完成）：对 CurrentChampion（停机）和 Antihydra（未知）运行信息状态格构造，比较 ρ(J)。

### B-24.4.3　判据的定义

**定义 B-24.4.2（信息状态极限谱半径）**：对图灵机 M，定义：

ρ^{info}_∞(M) := lim_{L→∞} ρ(J(B^{info}_{M,L}))

其中 L 为窗口大小，同时也是信息状态向量的窗口参数。

**判据 B-24.4.1（信息状态停机判据）**：

- 若 ρ^{info}_∞(M) = 0，则 M 在 ISD 框架下被判定为停机。
- 若 ρ^{info}_∞(M) > 0，则 M 在 ISD 框架下被判定为非停机。

**定理 B-24.4.3（判据的单调收敛性）**：ρ^{info}_∞(M) 的定义是良定义的：序列 ρ_L = ρ(J(B^{info}_{M,L})) 单调有界，因此极限存在。[⟹] 从 B-24.4.1 和 v3.44。

**证明**：

1. **有界性**：由定理 B-24.4.1，0 ≤ ρ_L < 1 对所有 L 成立。
2. **单调性**：当 L 增大时，信息状态向量包含更多窗口信息（更长的范围），分箱向量集合单调不减（新向量加入，旧向量保留）。因此 B^{info}_{M,L} ⊆ B^{info}_{M,L+1}（作为格嵌入）。由 v3.44（链长不变性推广），格扩张使谱半径单调变化。
3. 有界 + 单调 → 极限存在。∎

**注**：单调性严格证明需要补充论证格扩张对谱半径的单调性影响——当前标记为 ◇ 部分证明。

### B-24.5.2　参数选择

| 参数 | 推荐值 | 理由 |
|------|--------|------|
| 模拟步数 N | 50000 或直到循环 | 足够捕获行为模式 |
| 窗口 L | 5, 10, 20, 50, 100 | 多尺度扫描 |
| 分箱数 B | 4 | 平衡分辨率和格大小 |

### B-24.5.3　预期格大小

取 B = 4, d = 10，理论上界 B^d = 1,048,576。实际中：
- 混沌轨迹：预计覆盖数百个分箱向量 → 概念数 ~100-500
- 周期轨迹：预计覆盖数十个分箱向量 → 概念数 ~10-50
- 这远大于 v1/v2 的 2-4 概念，足以使 N 算子产生可区分的 ρ(J)

## B-24.6　理论后果

### B-24.6.1　与 v1/v2 的关系

**定理 B-24.6.1（信息状态格避开退化定理）**：B-24.3.1 的退化定理不适用于 B^{info}_{M,L}，因为其对象不是配置节点而是时间步，其属性不是位置-状态三元组而是信息状态的分箱。[⟹] 从 B-24.3.3。

**证明**：退化定理的证明依赖于"确定性图灵机配置图出度 ≤ 1"。新构造中，对象是时间步，其"出度"概念不存在——时间步之间没有转移关系，只有相似性关系（由分箱向量决定）。因此退化定理的前提不成立。∎

### B-24.6.2　ISD 框架的适用范围（更新）

**定理 B-24.6.2（ISD 适用范围更新定理）**：信息状态格构造将 ISD 框架的适用范围从"概念层次结构"扩展到"信息状态轨迹结构"。N 算子的区分能力不再依赖于格的"概念层次丰富度"，而是依赖于格的"信息状态多样性"——即轨迹覆盖的分箱向量数量。[⟹] 从 B-24.3.2 和 B-24.4.2。

**推论 B-24.6.1**：信息状态格构造理论上有能力区分任何产生不同信息状态轨迹的 TM。对于 TM 停机问题，这等价于：若停机和不停机的信息状态轨迹具有不同的分箱向量覆盖，则 ISD 可以区分。
