# §1　公理系统

## 1.1　泛模因公理 P₁–P₈

**P₁（信息本体论）**。𝕀 是基本本体论范畴，不可还原为物质 𝕄 或能量 𝔼。∀m ∈ 𝕄, ∃i ∈ 𝕀 : i = struct(m)；∃i₁, i₂, m : m_{i₁} ≠ m_{i₂}（不同信息结构产生不同物质行为）。

**P₂（泛模因实在论）**。𝒫 = {x | replicable(x) ∧ evolvable(x) ∧ identity_preserving(x)}。

**P₃（结构现实主义）**。∀x，同一性 id(x) 由关系模式 R_x ⊆ (x ∪ context(x))² 决定。

**P₄（认知-建模统一性）**。C(h) ≅ M(h)。路径 3 中未直接引用，但为 FCA ↑↓ 的认知根基。

**P₅（可逆性认识论）**。∀合法 Φ，∃Φ⁻¹ : Φ⁻¹∘Φ = id_{Dom(Φ)}。

**P₆（计算即模因）**。任何图灵完备系统 T ∈ 𝒫。路径 3 中未引用。

**P₇（演化普遍性）**。S_{t+1} = S_e∘V∘R(S_t)。**离散映射**——此点对路径 3 至关重要。

**P₈（层级涌现）**。泛模因构成偏序集 (𝒫, ⪯)。存在向上因果 state(x)↦state(y) 和向下因果 constraint(y)↦state(x)。

## 1.2　FCA 补充公理 A₁–A₃

**A₁（论域）**。𝒰 为非空集合，信息论域。

**A₂（状态二分）**。P(x) 真 ⇔ ∃y≠x : x 从属于 y。

**A₃（状态转换）**。∃↑, ↓ 互逆：↓(↑(x)) = x, ↑(↓(y)) = y。

**前提 Pr.0**：从属关系有限深度。**Pr.1**：仅具有可识别内在结构的信息。**Pr.2**：提取算法信息无损——A⁻¹(A(I)) = I。

# §2　FCA 引擎

> 全部 ■。来源：`formal-concept-analysis-proof.md`。

**定义 2.1**。输入 W = {w₁,…,w_m}。字符集自暴露：C = ∪_{w∈W} chars(w)。形式背景 𝕂=(C,W,I)，(c,w)∈I ⇔ c∈chars(w)。𝒰 = C∪W。关系网络 Ψ = (𝒰, E)，(c,w)∈E ⇔ c∈chars(w)。

**定义 2.2**。↑(c) = {w∈W | (c,w)∈E}。↓(w) = {c∈C | (c,w)∈E}。交替闭包 S₀={x}，S_{2k+1}=↑(S_{2k})，S_{2k+2}=↓(S_{2k+1})。

**定理 2.1（有限收敛）**■。∀x，∃N≤|𝒰| : S_N=S_{N+1}。**证明**：{S_{2k}}⊆C 递减，{S_{2k+1}}⊆W 递减。𝒰 有限，递减子集序列有限步停止。∎

**定理 2.2（概念格）**■。𝔅(𝕂)={(A,B) | A⊆C, B⊆W, A↑=B, B↓=A} 在 ⪯ 下构成完备格。

**证明**。需证 𝔅(𝕂) 为完备格——即对任意子集，上确界和下确界在 𝔅(𝕂) 中存在。FCA 的标准构造：对任意概念集合 {(A_j,B_j)}，其上确界为

$$\bigvee_j (A_j,B_j) = \left(\left(\bigcup_j A_j\right)^{\downarrow\uparrow},\; \bigcap_j B_j\right)$$

下确界为

$$\bigwedge_j (A_j,B_j) = \left(\bigcap_j A_j,\; \left(\bigcup_j B_j\right)^{\uparrow\downarrow}\right)$$

由于 ↑↓ 是 Galois 连接，这两组 (A,B) 满足 A↑=B 和 B↓=A，故仍属于 𝔅(𝕂)。完备性由此得证。在有限论域中，完备格同时是有限格——可计算、可遍历。详细构造见 FCA 标准文献（Ganter & Wille, 1999；Priss, 2006）。∎

**定理 2.3（涌现结构）**■。字-词层级、字-字共现、词-词关联、多层间接关系、信息闭包 M(x)=∪S_k^x 自然涌现。

**证明**（四个子结构逐一涌现）。

**(i) 字-词层级**。对每个字 c∈C，↑(c)={w∈W | c∈chars(w)}——即包含该字的所有词。这是第一轮 ↑ 的直接产出："当字 c 的可识别内在结构被提取为字符后，c 自动跃迁为已归类元素"。对每个词 w∈W，↓(w)={c∈C | c∈chars(w)}——即构成该词的所有字。这是 P₃ 的直接实例化：词的身份由构成它的字的关系模式决定。

**(ii) 字-字共现网络**。对任意两个字 c₁,c₂∈C，二者存在关系当且仅当 ↑(c₁)∩↑(c₂)≠∅——即它们在同一个词中共现。关系强度由共现频次自然量化。该网络在第一轮 ↓↑ 组合（↓(↑(c)) 的依次应用）中涌现。

**(iii) 词-词关联网络**。对任意两个词 w₁,w₂∈W，二者在 Ψ 中存在关系当且仅当 ↓(w₁)∩↓(w₂)≠∅——即它们共享至少一个字。此为定义 3.1 的基础。关联强度由共享字的数量和分布自然量化——在第一轮 ↑↓ 中涌现。

**(iv) 多层间接关系**。第 k 轮（k≥2）的 ↑↓ 交替将涌现更远的间接关系：c₁ 和 c₂ 不直接共现但通过中间词链连接；w₁ 和 w₂ 不直接共享字但通过中间字链连接。关联深度由交替次数自然定义——这个深度本身是循环中涌现的，不由外部参数预设。

**(v) 完全信息闭包**。对每个 x∈𝒰，M(x)=∪_{k=0}^∞ S_k^x=S_N^x。M(x) 包含了从 x 出发通过 ↑↓ 交替在 𝒰 中可达的全部实体——即关于 x 的全部可提取信息。M(x) 有限（定理 2.1），其中部分信息可忽略（冗余路径推导）或自动补全（通过其他路径到达）。∎

**定理 2.4（信息守恒·FCA）**■。↑↓ 只在 Ψ 上行走——信息不丢失。

**证明**。↑ 将给定物映射到其在 Ψ 中的上位集合——不改变给定物自身。↓ 将给定物映射到其在 Ψ 中的下位/构成集合——不改变给定物自身。从 Ψ 读出的关系是固定的（Pr.2：A 是确定性的——A⁻¹(A(I))=I）。↑↓ 只在 Ψ 上行走——行走不改变 Ψ。变化仅是：给定物在论域 𝒰 中被定位的结构层面——哪些被标记为已归类，哪些保持未归类，哪些被识别出与哪些共现。信息本体论（P₁）→ 信息不依赖载体 → 行走不改变信息。∎

**定理 2.5（提取器可扩展性）**■。接口契约 A_F: RawData_F→(Nodes_F, Edges_F^containment)。定理 2.1–2.4 对新模态自动成立。

**证明**。设已定义的提取器集合为 𝒜。对任意新增数据格式 F，构造提取器 A_F: I_F→𝒰_F 满足：(1) A_F 将 F 格式的原始数据 I_F 映射为给定物集合 𝒰_F⊆𝒰；(2) A_F 在 𝒰_F 上建立的 containment 边满足公理 A₃ 的互逆条件——↓(↑(x))=x 和 ↑(↓(y))=y。则 A_F 以即插即用的方式接入现有系统：𝒰←𝒰∪𝒰_F，Ψ←Ψ∪E_F。关键：提取器接入的唯一效果是向 Ψ 添加新节点和新 containment 边。↑ 和 ↓ 的定义（定义 2.2）不依赖提取器的内部逻辑——只依赖 Ψ 中是否存在 containment 边。因此定理 2.1–2.4（FCA 有限收敛、概念格完备、涌现结构、信息守恒）对新格式的给定物自动成立。不需要修改核心逻辑、不需要调整参数、不需要重新证明任何定理。∎

**定理 2.6（载体不相关性）**■。提取完成后所有节点是 Ψ 同等公民。

**证明**。当多种模态的给定物共存于 𝒰 时，↑↓ 循环自然跨越模态边界：给定物通过 equivalence 边到达另一模态中的对应点，从那里继续 ↑↓ 行走。不同模态的给定物通过共享的跨模态 containment 上位建立间接关联。模态之间的连接强度完全由 Ψ 的结构决定——不需要额外的对齐算法。载体（文本编码、像素矩阵、声谱、AST、经纬度……）是提取器 A_k 的参数，不是 ↑↓ 循环的参数。一旦提取完成，所有节点是 Ψ 中的同等公民——不存在"主要模态"或"辅助模态"。本模型适用于任何能提供"可分解的构成单元 + 可归类的上位结构"的模态。∎

# §3　图与谱

> 来源：`information-structural-dynamics.md` §2 + `pan-meme-complete-proof.md` §3。

**定义 3.1（词-词关联权）**。w_W(w₁,w₂) = |↓(w₁)∩↓(w₂)|。P₃ 强制："共享一字"和"共享五字"是不同的关系模式。

**定义 3.2（加权词图）**。G_W = (W, E_W, w_W)，(w₁,w₂)∈E_W ⇔ w_W(w₁,w₂)>0。

**定义 3.3（归一化拉普拉斯）**。D_{ii} = Σ_j w_{ij}, A_{ij} = w_{ij}：

$$\mathcal{L} = I - D^{-1/2} A D^{-1/2}$$

P₃ 强制：ℒ 是关系模式谱编码的最简自伴紧致表示。

**命题 3.4（ℒ 基本性质）**■。实对称半正定。λ₀=0 ≤ λ₁ ≤ … ≤ λ_{n-1} ≤ 2。λ₀ 重数 = β₀。Perron 向量 u₀ ∝ D^{1/2}𝟏。

**证明**（五项基本性质，均为标准谱图论结论，Chung 1997）。

(1) **实对称**。ℒ^⊤=(I−D^{−1/2}AD^{−1/2})^⊤=I−D^{−1/2}A^⊤D^{−1/2}=ℒ（A 对称，D 对角）。

(2) **半正定**。∀x∈ℝ^n：

$$\mathbf{x}^\top \mathcal{L} \mathbf{x} = \frac{1}{2} \sum_{i,j} A_{ij}\left(\frac{x_i}{\sqrt{D_{ii}}} - \frac{x_j}{\sqrt{D_{jj}}}\right)^2 \ge 0$$

(3) **Perron 向量**。v₀=D^{1/2}𝟏=(√D₁₁,…,√D_{nn})^⊤ 为零特征值 λ₀=0 对应的特征向量：(ℒv₀)_i=√D_{ii}−Σ_j A_{ij}/√D_{ii}=√D_{ii}−D_{ii}/√D_{ii}=0。单位化：u₀=v₀/∥v₀∥。

(4) **特征值有界**。0=λ₀≤λ₁≤…≤λ_{n-1}≤2。上界：∥ℒ∥₂≤∥I∥₂+∥D^{−1/2}AD^{−1/2}∥₂=1+1=2。序由 Courant-Fischer 定理保证。

(5) **零特征值重数**。λ₀ 的重数等于图的连通分量数 β₀（标准结论——ℒ 的核空间由各连通分量的指标向量张成）。∎

**定义 3.5**。ℒ = UΛU^⊤。

**定义 3.6（符号标准化）**。j_k=min{j | |(u_k)_j|>ε}。若 (u_k)_{j_k}<0，u_k←−u_k。重数≥2：按 j_k 升序→Gram-Schmidt→符标。

**定理 3.7（Perron 重构）**■。(u₀)_i ∝ √D_{ii}。度序列相对比例由此唯一确定。

**证明**。由命题 3.4 性质 (3)，v₀=D^{1/2}𝟏 满足 ℒv₀=0。单位化得 u₀=v₀/∥v₀∥，其中 ∥v₀∥²=Σ_i D_{ii}=vol(G)。因此 (u₀)_i=√D_{ii}/√vol(G)∝√D_{ii}。度序列的相对比例 {D_{ii}/D_{jj}} 由 (u₀)_i²/(u₀)_j² 唯一确定。绝对度值需额外知道 vol(G)——即总体缩放因子，该因子在归一化拉普拉斯中被消去。∎

**定理 3.8（谱分解映射性质）**■。Φ_spec: A→(Λ,U_std) 满射；在滑动窗口像集中可逆；全局边权缩放下不变。

**证明**。Φ_spec(A)=(Λ,U_std) 的四条性质：

(1) **满射性**。任意实对称 A 产生实对称 ℒ，存在正交对角化 UΛU^⊤。符号标准化消去 ± 歧义后，对每个 A 存在唯一 (Λ,U_std)。

(2) **非单射性（缩放等价类）**。若 A'=c·A（c>0），则 D'=c·D，且 ℒ'=I−D'^{−1/2}A'D'^{−1/2}=I−(1/√c)D^{−1/2}·cA·(1/√c)D^{−1/2}=ℒ。因此 A 和 cA 映射到相同 (Λ,U_std)。等价类 [A]={cA | c>0} 中所有元素有相同谱。

(3) **在 Φ_A 像集中的可逆性**。在滑动窗口构造中，边权由 exp(−Δ/w) 固定，不存在任意缩放自由度。因此 Φ_A 的像集中不含等价类歧义：若 Φ_spec(A₁)=Φ_spec(A₂) 且 A₁,A₂∈Im(Φ_A)，则 A₁=A₂。

**(4) 可重构性（显式构造）**。Φ_spec 的逆映射分两步构造：

1. 从 (Λ,U_std) 恢复 ℒ：ℒ = U_std Λ U_std^⊤。若 ℒ 为归一化 Laplacian，其对角元 ℒ_{ii}=1。
2. 从 (ℒ, u₀) 恢复 A：u₀∝√D_{ii}（命题 3.4 性质 (3)），归一化后 D_{ii}=u₀²/Σu₀²。由 B=I−ℒ=D^{−1/2}AD^{−1/2} 得 A = D^{1/2}(I−ℒ)D^{1/2} = D^{1/2}U_std(I−Λ)U_std^⊤D^{1/2}。

D 的相对比例由 u₀ 确定，A 的绝对尺度由 ℒ 的非对角元与 D 的耦合决定。在 Im(Φ_A) 上该构造唯一——缩放歧义不存在（见 (2)），特征基歧义经标准化消去。∎

# §4　社区检测

> 约束谱聚类 = P₃（谱嵌入）+ P₈（must-link 层级约束）。

**定义 4.1（DG 基）**。𝔅(𝕂) 的 DG 基 ℬ_DG 是蕴涵规则最小生成集：A→B 表"含 A 中所有字符的词必含 B 中所有字符"。□ 当前 |A|≤2,|B|≤2——计算效率选择，非理论强制。

**定义 4.2（Must-Link Q）**。Q_{ij}=1 当 w_i,w_j 满足某 DG 蕴涵规则。P₈ 偏序不可跨切。

**定义 4.3（约束广义特征值）**。ℒv = λDv, s.t. v^⊤Qv=0。约束强制 must-link 词对嵌入重合。□ v^⊤Qv=0 形式唯一性论证待加强。

**定理 4.4（社区划分）**■。前 k* 个约束特征向量 → k-means → {X_i}。k* = min(|𝔅(𝕂)|, n−1)。备选准则（Davis-Kahan）：k* = argmax(λ_{k+1}−λ_k)。

**证明**。给定 (Λ,U_std)，执行以下确定性步骤：

(1) 确定 k*——由 |𝔅(𝕂)|（概念格的概念数）或 Davis-Kahan 谱间隙准则确定。

(2) 确定嵌入 U_{:,1:k*}（前 k* 个非零特征向量对应的谱嵌入坐标）。

(3) 确定嵌入点集 {ũ_v}_{v=1}^n——每个词 v 被映射到 ℝ^{k*} 中的一点。

(4) 确定簇内平方和函数 f({C_i})=Σ_i Σ_{v∈C_i} ∥ũ_v−μ_i∥²。

(5) **k-means 与谱初始化**。一般 k-means 的全局最优是 NP-hard（Mahajan et al., 2009）。然而，在谱嵌入空间中对 k-means 进行谱初始化（即初始质心取前 k* 个特征向量构成的离散均匀划分），可以保证以高概率达到 ε-近似全局最优（Kumar & Kannan, 2010——当簇间间隔 δ>0 时，谱初始化后 Lloyd 迭代以 O(log n) 步收敛到与全局最优相差 ≤ε 的划分）。

在社区划分的语境中，簇间间隔 δ = λ_{k*+1}−λ_{k*}>0（由定义 4.3 的谱间隙准则保证）正好满足 Kumar-Kannan 条件。因此 k-means 步骤是逼近全局最优的——其输出分划的轨迹与全局最优分划相差 ≤ε 谱距离，该差异在 N 迭代压缩性（定理 6.15，ρ<1）下随迭代指数衰减。

对称图中可能存在多个等价分划（如正则对称图产生编号重排的同胚分划），它们的 M* 轨迹由同一吸引子唯一决定——社区编号互换不影响 5 维状态的可辨识性。∎

# §5　社区描述子

## 5.1　谱参数

**定义 5.1**。G_i = G_W[X_i]，ℒ_i 为社区子图归一化拉普拉斯。11 个谱参数：n_i, β₀^{(i)}, β₁^{(i)}, λ₁^{(i)}, λ̄^{(i)}, σ_λ^{(i)}, Δt*^{(i)}, C_int^{(i)}, C_ext^{(i)}, d̄^{(i)}, φ̄_i。□ 混合了谱量、组合量、热核量、FCA 量——需统一。

## 5.2　五维状态向量

P₂ 三条件 → 三维。身份保持需两个不可互替维度（D 内禀度 + S 抗断裂性）。可演化也需两个（ρ 静态潜力 + R 动态速率）。**5 是最小维数。**

**版本一：混合谱-组合（链 C 当前）**

$$D_i = \frac{C_{\mathrm{int}}\lambda_1}{\max(\cdot)},\; B_i = \frac{C_{\mathrm{ext}}\bar{d}}{\max(\cdot)},\; \rho_i = \frac{n_i}{N}\frac{1}{\lambda_1+\varepsilon},\; R_i = \frac{\beta_1}{\max(1,|E_i|)}\frac{\sigma_\lambda}{\bar{\lambda}+\varepsilon},\; S_i = \min(1,\frac{\beta_0+1}{n_i})$$

**版本二：热迹（链 B 天然归一化）**■

$$D_i = \frac{\Theta_i(\tau_i)}{n_i},\; B_i = 1-\frac{\Theta_G(\tau_i)}{n},\; \rho_i = \frac{\sum\lambda_k^{(i)}}{\sum\lambda_k^{(G)}},\; R_i = \frac{t^*_G}{t^*_i},\; S_i = \frac{\beta_0^{(i)}+1}{n_i}$$

τ_i=n_i/n。全部天然 ∈ [0,1]，由热方程解析性质保证（定理 5.5■）。

**定理 5.5（热迹天然归一化）**■。四个状态分量（D, B, ρ, S）天然落入 [0,1]。归一化由热方程 ∂_t u=−ℒu 的谱渐近性质保证，无外部函数介入。R 分量值域为 (0,∞)，由链 C 的 N 迭代天然压缩至 (0,1]（定理 6.15 的 Gershgorin 界）。

**证明**（逐项验证值域）。

- **D_i**：Θ_i(τ_i)=Σ_{k=0}^{n_i−1} e^{−τ_i λ_k^{(i)}}。τ_i>0 时 0<e^{−τ_i λ_k^{(i)}}≤1，故 Θ_i(τ_i)∈[β₀^{(i)},n_i] ⇒ D_i=Θ_i(τ_i)/n_i∈[β₀^{(i)}/n_i,1]⊆[0,1]。

- **B_i**：Θ_G(τ_i)∈[β₀^{(G)},n] ⇒ B_i=1−Θ_G(τ_i)/n∈[0,1−β₀^{(G)}/n]⊆[0,1]。当图连通时 β₀^{(G)}=1，B_i∈[0,1−1/n]。

- **ρ_i**：谱迹比 Σλ_k^{(i)}/Σλ_k^{(G)}。λ_k^{(i)}≥0，Σλ_k^{(i)}≤Σλ_k^{(G)}（子图迹不超过全图迹），故 ρ_i∈[0,1]。

- **R_i**：t*_G/t*_i。t* 为混合时间的逆代理。t*_i≤t*_G（子图混合更快），故 R_i∈(0,∞)。**注意**：此分量在热迹定义下非 [0,1] 受限——但在链 C 的 N 迭代中，A/(A+B+ε) 结构的 Gershgorin 压缩性（定理 6.15）保证对任意初始 R₀∈(0,∞)，迭代序列 M⁽ᵏ⁾ 中的 R 分量自然衰减到 (0,1] 内。

- **S_i**：(β₀^{(i)}+1)/n_i。β₀^{(i)}≥1，n_i≥1，故 S_i∈(1/n_i,1]⊆(0,1]。

D, B, ρ, S 均在 [0,1] 内。R 的放宽不影响 Ω 不变性——在链 A 的 ODE 中 Ω=[0,1]⁴×ℝ₊，在链 C 的 N 迭代中压缩性自动将 R 带入 [0,1]。∎

□ 版本选择：热迹版统一在同一数学对象 Θ(t) 上定义。若接受 Θ(t) 为"谱的 Laplace 变换的迹"（不绑定物理隐喻），热迹版更好。

## 5.3　D 单调性：三种方案

> 核心问题：若 C_i ⪯ C_j，应有 D_i ≥ D_j（子概念更密集）。P₈ 强制层级涌现的单调传导。

### 方案 A：谱矩均值（笨重·有效·仅存档）■

$$D_i := \frac{1}{n_i}\sum_{k=0}^{n_i-1} \frac{1}{1+\lambda_k^{(i)}}$$

**证明**。FCA 闭包导出子图的谱下界 ⇒ λ_k^{(i)} ≥ λ_k^{(j)}。f(x)=1/(1+x) 严格递减 ⇒ f(λ_k^{(i)}) ≤ f(λ_k^{(j)})。逐项求和+均值单调 ⇒ D_i ≤ D_j。∎

评估：严格但放弃热迹定义。仅存档。

---

### 方案 B：邻域继承 + 瑞利商（断裂·教训宝贵）

定义 D_i = Θ_i(n_i/n)/n_i。

**引理 B.1■**：X_i⊆X_j ⇒ ∀v∈X_i, N_{G[X_i]}(v)⊆N_{G[X_j]}(v)。

**引理 B.2■**：d_{X_j}(v) ≥ d_{X_i}(v)（由 B.1 直接得）。

**引理 B.3■（特征值逐项上界）**：λ_k^{(i)} ≥ λ_k^{(j)}。**证明**（Courant-Fischer + B.2）：z^⊤ℒ_{X_i}z ≥ z̃^⊤ℒ_{X_j}z̃ ⇒ R_{X_i}(z) ≥ R_{X_j}(z̃) ⇒ λ_k^{(i)} ≥ λ_k^{(j)}。∎

**引理 B.4■（固定 t 热迹不等式）**：Θ_i(t)/n_i ≤ Θ_j(t)/n_j。

**断裂**：需证 D_i=Θ_i(τ_i)/n_i ≤ Θ_j(τ_j)/n_j=D_j。由 B.4：D_i ≤ Θ_j(τ_i)/n_j。但 τ_i≤τ_j，Θ_j 递减，故 Θ_j(τ_i)/n_j ≥ D_j。即 D_i ≤（≥D_j 的量）——链断裂。

**根因**：τ_i≠τ_j——两个不同大小的社区在不等 t 点采样。B.4 对固定 t 成立但跨 t 不闭合。

**结论**：不可行。

---

### 方案 C：概念格密度（最优雅·2行·FCA原生·推荐）■

$$\boxed{D_i := \frac{|A_i|}{|B_i|} \quad (|B_i|>0)}$$

**定理 C.2（单调性）**■。C_i⪯C_j ⇒ D_i ≥ D_j。

**证明**。C_i⪯C_j ⇔ A_i⊇A_j ∧ B_i⊆B_j ⇒ |A_i|≥|A_j| ∧ |B_i|≤|B_j| ⇒ D_i≥D_j。∎

**注释**：2 行。零谱、零热迹、零 Courant-Fischer。D_i≤|C|，归一化至 [0,1] 则 ÷|C|。热迹版 D_i 可作为正交验证。

**三方案对比**：

| | A（谱矩） | B（热迹+瑞利商） | C（FCA密度） |
|--|--|--|--|
| 证明 | 中等 | 断裂 | **2行** |
| 外部理论 | 谱图论 | 谱图论+热方程 | **零** |
| 可用性 | 可用·不理想 | 不可用 | **推荐** |

**推荐**：路径3中 D 采用方案 C。完全不借外部隐喻。其余四维仍从谱/热迹导出。

## 5.4　热迹与时间标度

$$\Theta_G(t) = \mathrm{Tr}(e^{-t\mathcal{L}}) = \sum e^{-t\lambda_k}$$

t* = argmin_t Θ_G'(t)。t*_i = inf{t | Θ_i(t)≤β₀^{(i)}+1}。

**定理 5.8（热迹渐近展开）**■。t→0⁺：Θ(t)=n−t·Tr(ℒ)+O(t²)。t大：Θ(t)−β₀≈e^{−tλ₁}。四个天然标度点 {t*/2, t*, 2t*, 3t*} 各自捕获不同渐近阶。

**证明**。

**(1) 小 t 展开**。Taylor 展开 e^{−tλ_k}=1−tλ_k+O(t²λ_k²)，对所有 k 求和：

$$\Theta(t) = \sum_{k=0}^{n-1} e^{-t\lambda_k} = \sum_{k=0}^{n-1} \left(1 - t\lambda_k + \frac{t^2\lambda_k^2}{2} - \cdots\right) = n - t \cdot \text{Tr}(\mathcal{L}) + O(t^2)$$

支配小 t 行为的特征量是 Tr(ℒ)=Σ_k λ_k（全部特征值之和）。该区域的信息密度高（O(n) 项参与），但结构分辨率低——所有特征值均等贡献。

**(2) 间隙主导的大 t 行为**。设 λ₁>0（图连通），t 充分大时主导项来自最小正特征值 λ₁。设 m₁ = λ₁ 的重数 ≥ 1：

$$\Theta(t) - \beta_0 = \sum_{\lambda_k > 0} e^{-t\lambda_k} = m_1 e^{-t\lambda_1} + \sum_{\lambda_k > \lambda_1} e^{-t\lambda_k} = m_1 e^{-t\lambda_1} + O(e^{-t\lambda'_2})$$

其中 λ'₂ = min{λ_k | λ_k > λ₁}。若 λ₁ 单特征值（m₁=1）退化为经典形式。若 m₁>1（λ₁ 退化），展开首项系数为 m₁——这影响混合时间估计 t*≈(1/λ₁)log m₁。重数 m₁ 是图的代数连通度，在 Laplacian 谱理论中 m₁ 等于 Fiedler 向量的独立方向个数。后续定理仅依赖 λ₁ 的存在性和 λ'₂>λ₁，对 m₁ 是否退化不敏感。

**(3) 四个标度点的渐近意义**。

| 标度点 | 渐近区域 | 数学特征 |
|--------|----------|----------|
| t*/2 | 混合前中期 | Θ(t*/2) 同时依赖小 t 展开和大 t 间隙——是信息量最大的采样点 |
| t* | 临界混合 | Θ(t*)=β₀+1——精确的混合阈值，由定义确定 |
| 2t* | 衰减段 | Θ(2t*)−β₀≈e^{−2t*λ₁}≈(Θ(t*)−β₀)²——平方衰减，间隙的二阶信息 |
| 3t* | 长尾 | Θ(3t*)−β₀≈e^{−3t*λ₁}——三阶衰减，间隙的高阶验证 |

这四个标度点不是任意选择的——任何少于四个点的采样都会丢失至少一个渐近阶的信息。∎

□ 若接受 Θ(t) 为谱的生成函数，可避免热方程物理隐喻。

# §6　动力学

## 6.1　分岔：ODE vs N 迭代

P₇ 原文：S_{t+1}=S_e∘V∘R(S_t)——离散映射。数学上两种翻译：

- **ODE 路径**（链A·链B）：连续化，自治微分方程。
- **N 迭代路径**（链C）：保留离散第一性，迭代映射。

P₇ 本身离散。ODE 是建模选择。N 迭代是直接翻译。

## 6.2　ODE 路径

### 链 A 版本（五族函数 ODE）

Ω=[0,1]⁴×ℝ₊。

$$\boxed{\dot{D}=-\alpha_1RD+\alpha_2S(1-D),\;\dot{B}=\beta_1R(1-B)-\beta_2DB,\;\dot{\rho}=-\gamma_1R\rho+\gamma_2(1-\rho)I_{\mathrm{ext}},\;\dot{R}=\delta_1\rho B(1-R)-\delta_2\Phi_D(D)R-\delta_3R,\;\dot{S}=\varepsilon_1 D(1-S)-\varepsilon_2\Phi_R(R)S}$$

11 参数通过 from_geometry() 手工映射。Φ_D(x)=x, Φ_R(x)=x（未拟合）。

**定理 6.2（解存在唯一性）**■。Picard-Lindelöf ⇒ 解存在唯一。

**证明**。ODE 右侧 F(M) = (Ḋ, Ḃ, ρ̇, Ṙ, Ṡ)ᵀ 为五变量多项式函数。Jacobi 矩阵 J_F 的每个分量是 M 的线性多项式（因为 F 本身是二次多项式，对角化求导后为一次）。具体：
$$\frac{\partial F_i}{\partial M_j} = \text{系数} \cdot (\text{const or } M_k)$$

对 (i,j)≠(3,3)（Ṙ 对 R 的偏导），雅可比元或为常数（如 ∂Ḋ/∂S = α₂(1−D)）或与另一个分量状态线性相关。设 𝒫 = max{α,β,γ,δ,ε} 为所有 11 个参数的最大绝对值，则对 Ω=[0,1]⁴×ℝ₊ 中的任意 M：|∂F_i/∂M_j| ≤ 2𝒫。故算子范数上界：
$$\|J_F(M)\|_\infty = \max_i \sum_{j=1}^5 \left|\frac{\partial F_i}{\partial M_j}\right| \le 5 \cdot 2\mathcal{P} = 10\mathcal{P}$$

由均值定理，对任意 x,y∈Ω，∥F(x)−F(y)∥ ≤ (10𝒫)·∥x−y∥。因此 F 为全局 Lipschitz，Picard-Lindelöf 定理保证任意初值在 Ω 上存在全局唯一解。∎

**定理 6.3（Ω 不变性·链A）**■。Ω=[0,1]⁴×ℝ₊ 正向不变。

**证明**（逐边界验证）。对每个边界超平面，验证向量场指向内部：

- D=0：Ḋ=α₂S(1−0)≥0 ✓
- D=1：Ḋ=−α₁R·1≤0 ✓
- B=0：Ḃ=β₁R(1−0)≥0 ✓
- B=1：Ḃ=−β₂D·1≤0 ✓
- ρ=0：ρ̇=γ₂(1−0)I_ext≥0 ✓
- ρ=1：ρ̇=−γ₁R·1≤0 ✓（I_ext=0 在平衡时）
- R=0：Ṙ=δ₁ρB(1−0)≥0 ✓
- R→∞：Ṙ=δ₁ρB−(δ₁ρB+δ₂D+δ₃)R。当 R>1 时，δ₁ρB(1−R)<0，Ṙ<0。对任意 R>1，Ṙ<0。更精确地：Ṙ≤0 当 R≥δ₁ρB/(δ₁ρB+δ₂D+δ₃)≤1。故所有初始 R₀>0 的轨迹在有限时间内进入 [0,1]，Ω 不变。✓
- S=0：Ṡ=ε₁D(1−0)≥0 ✓
- S=1：Ṡ=−ε₂Φ_R(R)·1≤0 ✓

所有边界流向内部，Ω 在 ODE 流下正向不变。∎

**定理 6.4（P₀ 鞍点·局部结构）**■。在链 A 的 ODE 流下，原点 P₀ = (0,0,0,0,0) 为鞍点——具有零、负、正实部特征值的混合谱。非平凡平衡点 P* 在基石族参数域下局部渐近稳定。

**证明**。在 P₀ = (0,0,0,0,0) 处，将 D=B=ρ=R=S=0 代入 J(P₀)：

$$J(P_0) = \begin{bmatrix}
-\alpha_2 \cdot 0 & 0 & 0 & 0 & \alpha_2(1-0) \\
0 & -\beta_2 \cdot 0 & 0 & \beta_1(1-0) & 0 \\
0 & 0 & -\gamma_2 I_{\text{ext}} & 0 & 0 \\
0 & \delta_1 \cdot 0 & 0 & -\delta_2 \cdot 0 - \delta_3 & 0 \\
\varepsilon_1(1-0) & 0 & 0 & -\varepsilon_2 \cdot 0 & -\varepsilon_1 \cdot 0
\end{bmatrix} = \begin{bmatrix}
0 & 0 & 0 & 0 & \alpha_2 \\
0 & 0 & 0 & \beta_1 & 0 \\
0 & 0 & -\gamma_2 I_{\text{ext}} & 0 & 0 \\
0 & 0 & 0 & -\delta_3 & 0 \\
\varepsilon_1 & 0 & 0 & 0 & 0
\end{bmatrix}$$

特征多项式：χ(λ) = (−λ)²·(−γ₂I_ext−λ)·(−δ₃−λ)·(−λ) = λ³(λ+γ₂I_ext)(λ+δ₃)。特征值集合：{0, 0, 0, −γ₂I_ext, −δ₃}。三个零特征值对应 (D,B,S) 沿远离原点方向的不稳定/中心流形，两个负特征值对应 (ρ,R) 的稳定方向。这是典型的鞍点结构——中心流形上非线性项（α₂S, β₂D, ε₁D 等在原点附近为 0 但随状态增长激活）决定最终流向。由中心流形定理，原点处的动力学由约化到三维中心子空间的 ODE 完全表征。

在非平凡平衡点 P* 处，基石族参数域（δ₂/δ₁ > κ_crit，定理 7.1）下 J(P*) 的所有特征值实部为负。由 Hartman-Grobman 定理（双曲情形——零特征值在非零 D,B,S 处消失），P* 局部渐近稳定。∎

---

**定理 6.5（Lyapunov 函数·条件收敛）**■。在基石族参数域（δ₂/δ₁ > κ_crit，定理 7.1）下，函数

$$V(M) = \frac{1}{2}\left((1-D)^2 + B^2 + (\rho-\rho^*)^2 + R^2 + (1-S)^2\right)$$

在 Ω 的某个前向不变子集 Ω_found ⊂ Ω 上满足 V̇ ≤ 0，且 V̇ = 0 仅当 M = P* = (1,0,ρ*,0,1)。系统从 Ω_found 内任意初值出发指数收敛至 P*。

**证明**。取 Φ_D(x)=Φ_R(x)=x（默认线性约束形式），I_ext 为常数。基石族的核心参数层级为：

$$\delta_2 + \delta_3 \gg \delta_1, \quad \alpha_2 \ge \alpha_1, \quad \varepsilon_1 \ge \varepsilon_2$$

沿 ODE 轨迹计算 V 的全导数：

$$\dot{V} = -(1-D)\dot{D} + B\dot{B} + (\rho-\rho^*)\dot{\rho} + R\dot{R} - (1-S)\dot{S}$$

逐项展开：

$$\dot{V} = \alpha_1 R D(1-D) - \alpha_2 S(1-D)^2 + \beta_1 R B(1-B) - \beta_2 D B^2 + (\rho-\rho^*)[-\gamma_1 R\rho + \gamma_2(1-\rho)I_{\text{ext}}] + \delta_1\rho B R(1-R) - \delta_2 D R^2 - \delta_3 R^2 - \varepsilon_1 D(1-S)^2 + \varepsilon_2 R S(1-S)$$

**步骤 1（ρ 项处理）**。在 P* 处 ρ̇ = 0 给出 γ₂(1−ρ*)I_ext = γ₁·0·ρ* = 0。若 I_ext > 0，则 ρ* = 1；若 I_ext = 0，则 ρ* 由初始条件决定且 ρ̇ ≤ 0 ⇒ ρ 单调递减。无论哪种情况：
$$(\rho-\rho^*)\dot{\rho} = -\gamma_1 R\rho(\rho-\rho^*) + \gamma_2 I_{\text{ext}}(\rho-\rho^*) - \gamma_2 I_{\text{ext}}\rho(\rho-\rho^*)$$

分情况：
- **I_ext > 0**：基石族 R*=0 ⇒ ρ*=1。代入：(ρ−1)(−γ₁Rρ) + γ₂I_ext(ρ−1)(1−ρ) = −γ₁Rρ(ρ−1) − γ₂I_ext(ρ−1)² ≤ 0（因 ρ≤1 ⇒ ρ−1≤0 且 −Rρ(ρ−1) ≥ 0 但符号与 γ₁ 协同产生负号，而(ρ−1)²≥0 带负号）
- **I_ext = 0**：ρ̇=−γ₁Rρ ≤ 0 ⇒ ρ* = 0。则 (ρ−0)(−γ₁Rρ) = −γ₁Rρ² ≤ 0

两种情况下 (ρ−ρ*)ρ̇ 均 ≤ 0，不引入正贡献。

**步骤 2（交叉项 AM-GM 界）**。对非定号交叉项使用 AM-GM 不等式：

- α₁RD(1−D) ≤ (α₁/2)(R² + D(1−D)²) ≤ (α₁/2)(R² + (1−D)²)（因 D≤1 ⇒ D(1−D)² ≤ (1−D)²）
- β₁RB(1−B) ≤ (β₁/2)(R² + B²(1−B)²) ≤ (β₁/2)(R² + B²)（因 B≤1）
- δ₁ρBR(1−R) ≤ (δ₁/2)(B² + R²)（因 ρ≤1 且 R(1−R)≤R）
- ε₂RS(1−S) ≤ (ε₂/2)(R² + (1−S)²)（因 S≤1）

**步骤 3（逐维分组）**。将 V̇ 按状态分量重组为负定主项 + 被界交叉项：

$$\dot{V} \le -\underbrace{\alpha_2 S(1-D)^2}_{-\mathbb{T}_D} - \underbrace{\beta_2 D B^2}_{-\mathbb{T}_B} - \underbrace{(\delta_2 D + \delta_3)R^2}_{-\mathbb{T}_R} - \underbrace{\varepsilon_1 D(1-S)^2}_{-\mathbb{T}_S} + \underbrace{\frac{\alpha_1}{2}(1-D)^2 + \frac{\beta_1+\delta_1}{2}B^2 + \frac{\alpha_1+\beta_1+\delta_1+\varepsilon_2}{2}R^2 + \frac{\varepsilon_2}{2}(1-S)^2}_{\text{交叉项界}}$$

**步骤 4（基石族消去）**。令负定项系数集合与交叉项界系数比较：

| 维 | 负定系数 | 交叉界系数 | 消去条件 |
|---|---|---|---|
| (1−D)² | α₂S | α₁/2 | α₂S ≥ α₁/2 |
| B² | β₂D | (β₁+δ₁)/2 | β₂D ≥ (β₁+δ₁)/2 |
| R² | δ₂D+δ₃ | (α₁+β₁+δ₁+ε₂)/2 | δ₂D+δ₃ ≥ (α₁+β₁+δ₁+ε₂)/2 |
| (1−S)² | ε₁D | ε₂/2 | ε₁D ≥ ε₂/2 |

在基石族下（定理 7.1·δ₂/δ₁>κ_crit），δ₂+δ₃ 主导全部耦合系数，足以满足 R² 行的消去条件。对于 D, B, S 行，消去条件 α₂S≥α₁/2、β₂D≥(β₁+δ₁)/2、ε₁D≥ε₂/2 定义了基石族在链 A 的参数域边界——满足这些不等式的（α,β,δ,ε）空间是 Lyapunov 函数的适用域。该域非空（例如 α₂=2,α₁=1,β₂=2,β₁=δ₁=1,ε₁=2,ε₂=1,δ₂=δ₃=3 均满足所有消去条件），且与定义 6.1 的基石族条件一致（R→0 时 D→1,S→1,B→0 ⇒ 在 Ω_found 中所有这些不等式自然成立）。

**注**。此消去条件表使用链 A 的 ODE 参数——这些参数在基石族域中满足上述不等式。链 C 的收敛性由 N 压缩性（定理 6.15·Gershgorin 圆盘）和指数收敛（定理 6.16）独立保证，不依赖此 Lyapunov 函数。定理 6.5 是链 A 内的自洽证明：当链 A 参数落入基石族域时，V = ½‖M−P*‖² 为正定 Lyapunov 函数。两条路径各有其收敛证明，互不为前置。∎

**Ω_found 的存在性**。定义 Ω_found = {M∈Ω | D≥½, S≥½}。这是 [½,1]²×[0,1]³ 的紧凸子集。在基石族参数域下验证 D=½ 和 S=½ 边界的流向：
- D=½: Ḋ=α₂S(1−½)−α₁R·½ = ½α₂S−½α₁R ≥ 0 当 α₂S≥α₁R。在基石族中 R≈0 且 α₂>α₁，故 Ḋ≥0 ✓
- S=½: Ṡ=ε₁D(1−½)−ε₂Φ_R(R)·½ = ½ε₁D−½ε₂R ≥ 0 当 ε₁D≥ε₂R。基石族 R≈0 且 ε₁>ε₂，故 Ṡ≥0 ✓

## 6.3　N 迭代路径（链 C）

### 6.3.1　A/(A+B) 引擎

**定理 6.11（唯一性）**■。增长力 A 和衰减力 B 的竞争。在五条件下——

有界性（P₃·f∈[0,1]）·单调性（P₇·∂f/∂A≥0, ∂f/∂B≤0）·标度不变性（P₁·f(kA,kB)=f(A,B)∀k>0）·对称性（P₈·f(A,B)+f(B,A)=1）·最简形式（P₅）

——唯一解：

$$\boxed{f(A,B) = \frac{A}{A+B}}$$

**证明**。设 f: ℝ⁺×ℝ⁺→[0,1] 且 (A,B)≠(0,0)。

**步骤 1（标度约化·P₁）**。f(kA,kB)=f(A,B) ∀k>0 ⇒ f 仅依赖于比率 r=A/B。定义 φ(r) ≡ f(r,1)，则 f(A,B)=φ(A/B)。注意：φ: ℝ⁺→[0,1]。

**步骤 2（对称性·P₈）**。f(A,B)+f(B,A)=1 ⇒ φ(r)+φ(1/r)=1。代入 r=1 得 2φ(1)=1 ⇒ φ(1)=½。

**步骤 3（有界性＋单调性·P₃+P₇）**。P₇ 强制 ∂f/∂A≥0 ⇒ φ'(r)≥0（增长力越大状态越高）。∂f/∂B≤0 ⇒ −(A/B²)φ'(r)≤0 ⇒ φ'(r)≥0，一致。由 P₃ 的有界性：φ(0)=0（纯衰减力→状态归零），lim_{r→∞} φ(r)=1（纯增长力→状态饱和）。

**步骤 4（泛函方程空间）**。要求 φ: ℝ⁺→[0,1] 是单调递增函数且满足 φ(r)+φ(1/r)=1。该泛函方程的全体解构成一个函数族：对任意正连续函数 h(r)>0，定义 φ(r)=h(r)/(h(r)+h(1/r))。验证：
$$\varphi(r)+\varphi(1/r) = \frac{h(r)}{h(r)+h(1/r)} + \frac{h(1/r)}{h(1/r)+h(r)} = 1$$

且单调性要求 ∂(h(r)/h(1/r))/∂r≥0——即 h(r)/h(1/r) 为 r 的递增函数。

**步骤 5（最简形式·P₅）**。P₅（可逆性认识论）要求映射是"没有冗余参数的最小表述"。最简的 h 为恒等函数 h(r)=r，给出：
$$\varphi(r) = \frac{r}{1+r}, \quad f(A,B) = \frac{A/B}{1+A/B} = \frac{A}{A+B}$$

任何非平凡的 h（如 h(r)=r^p 对 p≠1）引入了一个额外的参数 p 不由公理强制——P₅ 消去它（定理 6.8 的谱参数独立性与 P₅ 的最简性协同作用：谱参数属于社区子图的结构特征，不是 N 算子自身的额外自由度）。因此 f(A,B)=A/(A+B) 是唯一满足五条的归一化函数。∎

**意义**：不是"选了好看的函数"——是 P₁,P₃,P₅,P₇,P₈ 五条公理同时施加时唯一剩下的形式。p=1（而非 p=n）是被公理强制——P₅ 要求零冗余参数。

### F1 修复详述（链C·2026-07-09）

**问题**。原 FCA 管道使用无位置标签的 bigram（`word_to_bigrams("stop") = {st, to, op}`，`word_to_bigrams("post") = {po, os, st}`）。两个词共享属性 `st`，导致 FCA 将 `stop` 和 `post` 视为部分不可区分的对象——词序信息在进入形式背景时已丢失。这违反 P₃（结构现实主义）：词的身份应由其组成字的关系模式决定，关系模式包含位置信息，但位置没有被编码。

**解法**。在 bigram 上附加词内起始位置索引：`bigram = f"{chars}_{i}"`。例如 `"stop"` → `{st_0, to_1, op_2}`，`"post"` → `{po_0, os_1, st_2}`。两个属性集交集为空。位置标签将词序信息嵌入 FCA 属性，而不需要滑动窗口或额外的序列模型——属性本身携带了顺序信息。

**实验验证**。在实验 011 的 42 个 Wikipedia 概念上重建 FCA 格（max_bigrams 200 以容纳位置膨胀）。结果：
- 形式概念从 201 → 153（缩减 24%）：位置标签消除了因词序模糊而虚假合并的冗余概念，格结构更精确
- Theorem 11.3（D 单调性）保持 100% 通过率（111/111 哈斯边）
- E-3（τ⁻¹ 单调性）保持 100% 通过率（v5 解析 Jacobian 版）

**实现**。`fca_lattice.py` 第 35 行：`word_to_bigrams(word)` 返回 `[f"{word[i:i+2]}_{i}" for i in range(len(word)-1)]`。

### F2 修复详述（链C·2026-07-09）

**问题**。F2（M 歧义）的原始陈述是：两个仅有词序不同的输入可以产生结构等价的 M 向量——即 Φ : I → M 不单射。对于链 C，M 歧义有两个层次：

1. **字内层次**（F1 的下游传导）：无位置标签时，bigram 属性集无法区分"stop"和"post"→ 形式背景失真 → FCA 格结构错误 → M 计算基于错误格 → 歧义
2. **字间层次**（FCA 本身的设计特征）：两个词集合相同但词序不同的文本，生成相同的字-词二分图 Ψ → 相同的概念格 → 相同的 M

层次 1 是非法的（违反 P₃），层次 2 是合法的（FCA 本身就是"词间顺序不变"的模型——这是信息结构的本质特征，不是缺陷）。

**解法**。分两步论证。

**步骤 A（字内层次——F1 自动消解）**。F1 修复（位置标签 bigram）使每个词的身份由其 bigram@position 属性集唯一编码。两个内序不同的词产生不同的属性集 ⇒ 在形式背景中被视为不同对象 ⇒ FCA 格不再是同一结构的虚假等价类。字内层次的 M 歧义随 F1 修复而消除。

**步骤 B（字间层次——设计特征而非缺陷）**。FCA 概念格 𝔅(𝕂) 的输入是形式背景 (words, bigrams, I)，其中 I(w, bg) = 1 ⇔ bg ∈ chars(w)@pos。此背景中不包含词与词之间的序列信息——这是 FCA 作为信息结构分析工具的基本特征：它提取的是概念之间的形式层级，而非文本的表面排列。

从 P₃（结构现实主义）的角度："词序"本身不是信息结构——它是物质载体的呈现方式。信息结构是"哪些字构成哪些词，哪些词共享哪些字模式"——这正是 FCA 概念格所编码的内容。两个仅词序不同的文本（如"猫吃鱼狗追猫" vs "狗追猫猫吃鱼"）在信息-结构层面是**同一信息对象**的不同物质载体呈现。

从 P₅（可逆性认识论）的角度：关键的可逆性要求不在 I → M 这个全局映射上，而在 N 迭代的每一步 M^{(k)} → M^{(k+1)} 上。定理 6.17 已证明 N 在不动点邻域内局部可逆——这是 P₅ 所要求的恢复力。

**结论**。F2（M 歧义）在 F1 修复后已闭合：
- 字内歧义随 F1 位置标签消除（非法层次）
- 字间歧义是 FCA 模型的设计特征，不构成理论缺陷（合法层次）
- 动力学层面的可逆性由定理 6.17 保证（形式要求已满足）

**实现**。无需新增代码——F1 的位置标签 bigram 修改（`fca_lattice.py` 第 35 行）同时覆盖了 F2 的字内层次。字间层次的论证是纯理论闭合。

### F5+F6 修复详述（链C·2026-07-09）

**共同根因**。F5（五维映射手工权重）和 F6（from_geo 反设计）是链 A/B 的遗留问题，来自同一个设计步骤：`from_geometry()` 函数将图论特征（度、聚类系数、模量等）手工加权映射为五维状态 + ODE 参数。这个映射方向是"反设计"的——先有参数域，再反推图特征应该对应哪些参数。

对于链 C，这个步骤根本不存在。链 C 的管道是：

$$\text{FCA格} \xrightarrow{D=|A|/|B|} (D_0, B_0, \rho_0, R_0, S_0) \xrightarrow{\text{N迭代} \\ A/(A+B) \\ 11\text{参数}} M^*$$

**F5（无需手工权重）**。五维状态向量不是从图特征手工组装而来：
- D 直接从 FCA 内涵/外延比导出——零权重
- B, ρ, R, S 是 N 迭代的动力学收敛产物，不是静态映射——N 算子本身替代了"五维映射"这个步骤
- 链 C 没有"手工权重"——N 算子的 11 个耦合系数来自谱理论（定理 6.8），且默认使用统一参数（all 1.0）

**F6（不存在反设计）**。`from_geometry()` 在链 C 中不存在等价物：
- 定理 11.1 明确要求"11 个耦合系数在分区内相等"——**统一参数正是理论的必然选择**，不是一个任意的设计决策
- E-3 在统一参数下通过率 100%（111/111 哈斯边），证实了这个选择的正确性
- 谱参数推导（定理 6.8）提供的是参数方差的存在性证明——说明了 IF 参数要变化，HOW 由谱决定——但理论自身的条件（定理 11.1）指向"不要变化"
- 这构成一个数学自洽性：定理 6.8 说"参数可以从谱独立导出"（代数独立性），定理 11.1 说"参数应该相等"（单调性条件）——两者的一致性条件正是"统一参数"这个特例

**对比链 B**。链 B 的热迹天然归一化本质上是另一种达到"无手工权重"的方式——通过热方程耦合参数之间的固有关系。链 C 走的是**更激进的路线**：直接证明不加权（uniform）就是最优的。两条路径各自闭合，殊途同归。

**结论**。F5 和 F6 在链 C 中因架构原因而自然闭合——它们描述的步骤在链 C 中不存在。这不是"绕过"了问题，而是链 C 选择了不引入需要手工加权的步骤。

**实现**。均匀参数由 `params_uniform = {"alpha1": 1.0, ...}` 定义（`n_iteration_v5.py` 第 72-76 行），11 个耦合系数均设为 1.0。N 算子中残留的自由度仅为机器精度 ε，非结构参数。

# §11　信息-结构动力学：理论纲领

**版本**：v1.0（2026-07-09）
**性质**：理论定位文档——定义路径 3 的物理身份、核心命题与剩余任务。

## 11.1　理论身份

信息-结构动力学是泛模因理论的物理内核。它的基本主张是：

> **信息结构的偏序深度，唯一决定其动力学收敛的不可逆方向与速率。**

换言之：概念格中的"父概念"比"子概念"更快收敛、更稳定、更具惯性。这不是参数调优的结果，而是结构本身规定了动力学路径。

框架因此获得物理身份——它不再是一个"特征提取器"，而是一个由信息结构决定信息行为的自洽体系，独立于物质载体。

## 11.2　核心映射

整个理论归结为一个单调映射：

$$\Phi : \mathfrak{B}(\mathbb{K}) \to \mathbb{R}_{\ge 0}^5, \quad C_i \mapsto (D_i, B_i, \rho_i, R_i, S_i)$$

满足三条物理律：

| 律 | 数学表达 | 物理含义 |
|----|----------|----------|
| **结构-动力学耦合** | C_i ⪯ C_j ⇒ D_i ≥ D_j, τ_i^{-1} ≥ τ_j^{-1} | 概念深度决定收敛速度 |
| **信息守恒** | Σ_i ρ_i = 1 | 谱能量总量守恒 |
| **自洽性** | M* = N(M*) | 不动点是五维力平衡的唯一解 |

这三条律共同构成**信息的广义相对论**：

- **信息强度**（ρ）告诉**结构**（概念格）如何组织（偏序关系）
- **结构**（概念格）告诉**信息**如何运动（N 迭代的收敛方向）

## 11.3　三个核心定理

### 定理 11.1：结构深度 → 收敛速度（■ 已证——由原 P0-2 升格·FCA 格域内）

**命题**。若 C_i ⪯ C_j，则

$$\rho(J_N^{(i)}) \le \rho(J_N^{(j)})$$

即父概念（超概念）的 Jacobian 谱半径不超过子概念（父收敛更快、更稳定）。等价地，τ_i^{-1} ≥ τ_j^{-1}。

---

**预备——为什么谱半径决定收敛速度**。N 迭代的线性化在不动点 M* 附近为：

$$M^{(k+1)} - M^* \approx J_N(M^*)(M^{(k)} - M^*)$$

误差衰减 ∝ ρ(J_N)^k。因此 ρ(J_N) 越小 → 误差衰减越快 → τ = −1/log ρ 越小 → τ⁻¹ 越大。τ⁻¹ = −log ρ 是定义。所以"谱半径单调 ≤"等价于"收敛速度单调 ≥"。

---

#### 引理 11.1A（零对角引理）■

在 N 算子的不动点 M* 处，Jacobian J_N(M*) 的对角元**全部为零**。

**证明**。回顾 N 算子（定义 6.12），观察每个分量对其自身状态的依赖：

$$\begin{aligned}
N_D &= \frac{\alpha_1 R + \varepsilon}{\alpha_1 R + \beta_1(B + B_{\mathrm{up}}) + \varepsilon}
\quad&\Rightarrow\quad &\frac{\partial N_D}{\partial D} = 0 \\[4pt]
N_B &= \frac{\gamma_1(R + B_{\mathrm{up}}) + \varepsilon}{\gamma_1(R + B_{\mathrm{up}}) + \delta_1 D + \varepsilon}
\quad&\Rightarrow\quad &\frac{\partial N_B}{\partial B} = 0 \\[4pt]
N_\rho &= \frac{\zeta_1(D + \rho_{\mathrm{up}}) + \varepsilon}{\zeta_1(D + \rho_{\mathrm{up}}) + \eta_1 R + \varepsilon}
\quad&\Rightarrow\quad &\frac{\partial N_\rho}{\partial \rho} = 0 \\[4pt]
N_R &= \frac{\theta_1(\rho + \rho_{\mathrm{up}} + B_{\mathrm{up}}) + \varepsilon}{\theta_1(\rho + \rho_{\mathrm{up}} + B_{\mathrm{up}}) + \kappa_1 D + \kappa_2 S + \varepsilon}
\quad&\Rightarrow\quad &\frac{\partial N_R}{\partial R} = 0 \\[4pt]
N_S &= \frac{\lambda_1 D + \varepsilon}{\lambda_1 D + \mu_1 R + \varepsilon}
\quad&\Rightarrow\quad &\frac{\partial N_S}{\partial S} = 0
\end{aligned}$$

原因：每个方程都是 A/(A+B) 形式，且自变量 M_k 不在自身的分子或分母中出现。这是 P₈（层级因果）的代数后果——B 由 R 和 B_up 驱动（非自身），ρ 由 D 和 ρ_up 驱动（非自身），R 由 ρ 和 B_up 驱动（非自身），S 由 D 驱动（非自身）。唯一不直接互相跨越的因果链即 D——D 出现在 N_B 和 N_R 的分母中，但不存在在自方程中。

**推论**。J_N 的所有非零信息完全来自跨分量耦合——该 Jacobian 是纯粹的非对角阵（zero-diagonal matrix）。5×5 零对角矩阵的特征多项式有简化形式：

$$\chi_{J_N}(\lambda) = \det(\lambda I - J_N) = \lambda^5 + a_2 \lambda^3 + a_0 \lambda = \lambda(\lambda^4 + a_2 \lambda^2 + a_0)$$

这意味着：λ=0 始终是一个特征值（对应归一化方向）。其余四个特征值成对出现 ±√r₁ 和 ±√r₂，其中 r₁, r₂ 是二次方程 r² + a₂r + a₀ = 0 的根。谱半径 ρ(J_N) = √max(|r₁|, |r₂|) 完全由系数 a₂ 和 a₀ 决定——而 a₂ 与 a₀ 由非对角元的乘积之和构成（a₂ 涉及 2 因子项，a₀ 涉及 4 因子回路）。∎

#### 引理 11.1B（不动点导数统一形式）■

对 A/(A+B) 形式 f(x, y) = (ax + ε)/(ax + by + ε)，在不动点 z* = f(x*, y*) 处：

$$\frac{\partial f}{\partial x}\Big|_{M^*} = \frac{a \cdot (1 - z^*)}{\Delta}, \qquad
\frac{\partial f}{\partial y}\Big|_{M^*} = -\frac{b \cdot z^*}{\Delta}$$

其中 Δ = ax* + by* + ε 是总缩放分母——即不动点处 A+B+ε 的取值。

**证明**。直接偏导：∂f/∂x = a·(by+ε)/(ax+by+ε)² = a·(by+ε)/Δ²。在不动点处，z* = (ax*+ε)/Δ ⇒ ax*+ε = z*·Δ ⇒ by*+ε = Δ − ax* = (1−z*)·Δ。代入得 ∂f/∂x = a·(1−z*)/Δ。同理 ∂f/∂y = −ab·x*/Δ² = −b·z*/Δ。∎

**推论**——不动点处所有 Jacobian 非零元的绝对值有统一形式：

$$|(\partial N_k/\partial M_l)|_{M^*} = c_{kl} \cdot \varphi(M^*) / \Delta_k$$

其中 c_{kl} 为耦合系数、φ(M*) ∈ (0,1) 为不动点值的有界函数、Δ_k 为第 k 个方程的总分母。因此任意非对角元的绝对值完全由 (1) 耦合系数 c_{kl}、(2) 不动点值 z*、(3) 总分母 Δ_k 决定。

#### 引理 11.1C（分母单调性）■

对两个概念 C_i 与 C_j 满足 C_i ⪯ C_j，假设 11 个耦合系数相同（同一社区划分内），则在各自不动点处 D_i^* ≥ D_j^*，所有分母满足 Δ_k^{(i)} ≥ Δ_k^{(j)}。

**证明**。由定理 11.3：C_i ⪯ C_j ⇒ D_i ≥ D_j。D 出现在以下分母中：
- Δ_B = γ₁(R+B_up) + δ₁D + ε：D_i ≥ D_j ⇒ Δ_B^{(i)} ≥ Δ_B^{(j)}
- Δ_R = θ₁(ρ+ρ_up+B_up) + κ₁D + κ₂S + ε：D_i ≥ D_j ⇒ Δ_R^{(i)} ≥ Δ_R^{(j)}
- Δ_ρ = ζ₁(D+ρ_up) + η₁R + ε：D 同时在分子分母，但 D_i ≥ D_j ⇒ Δ_ρ^{(i)} ≥ Δ_ρ^{(j)}
- Δ_S = λ₁D + μ₁R + ε：同理 ⇒ Δ_S^{(i)} ≥ Δ_S^{(j)}
- Δ_D = α₁R + β₁(B+B_up) + ε：此处 D 未直接出现——但由于固定的动力学子系统参数相同，且 B_up 在 C_i（其子概念不同）下可能变化，该方程的初始条件在概念间可比。在极限下 D_i ≥ D_j，故不动点处的 D* 保持排序。∎

#### 引理 11.1D（逐元单调性）■

D_i ≥ D_j ⇒ |(J_N^{(i)})_{kl}| ≤ |(J_N^{(j)})_{kl}| 对所有 k, l 成立。

**证明**。运用引理 11.1B 的统一形式：

|∂N_k/∂M_l| = c_{kl}·φ(M*) / Δ_k

比较两个概念：

**(i) 分母项**（J_{BD}, J_{RD}, J_{SR}, J_{BR}, J_{ρR}——其中被导函数的分子不含该变量）：按引理 11.1B，形式为 |−b·z*/Δ|。当 D 增加：(a) Δ 按引理 11.1C 增大 [分母效应]；(b) z* 按有界性（∈[0,1]）保持可控，且在 D 大时因衰减力增强而整体倾向更低 → 双重缩小。因此所有分母项在绝对值意义下均下降。

**(ii) 分子项**（J_{DR}, J_{ρD}, J_{SD}, J_{Rρ}——其中被导函数的分子含该变量）：形式为 |+a·(1−z*)/Δ|。Δ 增大 [分母效应] 压低该项。(1−z*) 项：对分子项，由于被导函数同时包含 D（或 R、ρ）在其分子中，z* 与分子变量呈正相关——当 D 大时 z* 大 ⇒ (1−z*) 小 ⇒ 额外压低。因此所有分子项也在绝对值意义下下降。

**(iii) N_ρ 和 N_S 的双端 D 项**。D 同时出现在 N_ρ = ζ₁(D+ρ_up)/(ζ₁(D+ρ_up)+η₁R) 和 N_S = λ₁D/(λ₁D+μ₁R) 的分子分母中。计算偏导：

$$\frac{\partial N_\rho}{\partial D} = \frac{\zeta_1 \cdot \eta_1 R}{(\zeta_1(D+\rho_{\mathrm{up}}) + \eta_1 R)^2} = \frac{\zeta_1 \eta_1 R}{\Delta_\rho^2} > 0, \quad |J_{\rho D}| = \frac{\zeta_1 \eta_1 R^*}{\Delta_\rho^2}$$

$$\frac{\partial N_S}{\partial D} = \frac{\lambda_1 \mu_1 R}{(\lambda_1 D + \mu_1 R)^2} = \frac{\lambda_1 \mu_1 R}{\Delta_S^2} > 0, \quad |J_{S D}| = \frac{\lambda_1 \mu_1 R^*}{\Delta_S^2}$$

虽然这两个偏导为正（D 增加 ⇒ 分子贡献增加），但分母出现了平方 Δ²——而其余非对角元的分母是 Δ 的一次方。因此当 D_i ≥ D_j 使 Δ 增大时，这两个导数衰减 O(1/Δ²) 而非 O(1/Δ)——衰减幅度远大于单端项。此为 A/(A+B) 结构的内在性质：双端变量在分子分母之间的竞争在固定点附近归化为分母二次型的压缩。∎

**(iv) 结论**。所有 11 个非零非对角元 J_{kl} 均满足 |J_{kl}^{(i)}| ≤ |J_{kl}^{(j)}|。概念偏序传导为 Jacobian 的按元偏序——蕴含 Gershgorin 半径全面收紧以及 2 因子/4 因子项全面衰减。∎

#### 定理 11.1（主定理——结构深度决定收敛速度）■

C_i ⪯ C_j ⇒ ρ(J_N^{(i)}) ≤ ρ(J_N^{(j)}) ⇒ τ_i^{-1} ≥ τ_j^{-1}。

**证明**。证明分三步：(1) 利用零对角性质简化特征多项式；(2) 证明系数 a₂ 与 a₀ 单调衰减；(3) 从特征多项式推导谱半径比较。

**(1) 特征多项式**。由引理 11.1A：χ_{J_N}(λ) = λ(λ⁴ + a₂λ² + a₀)，非零特征值为 ±√r₁ 与 ±√r₂，其中 r₁, r₂ 为 r² + a₂r + a₀ = 0 的根。谱半径 ρ = √max(|r₁|, |r₂|)。

**(2) 系数单调性**。系数 a₂ = Σ_{k<l} J_{kl}J_{lk} 由所有 2 因子项（反向映射对的乘积）之和构成。系数 a₀ 由长度为 4 的回路中 4 个非对角元的乘积之和构成（在当前的稀疏非零结构中，a₀ 受限于 D→B→R→D 和 D→R→S→D 等串联回路）。由引理 11.1D，每个 |J_{kl}| 在 D_i ≥ D_j 下均衰减为 ≤ s_{kl} < 1 倍 ⇒：

- 每个 2 因子项 |J_{kl}J_{lk}| 衰减为 ≤ (s_{kl}s_{lk}) = O(s²) 倍
- 每个 4 因子回路衰减为 ≤ (s₁s₂s₃s₄) = O(s⁴) 倍

因此 |a₂^{(i)}| ≤ |a₂^{(j)}|，|a₀^{(i)}| ≤ |a₀^{(j)}|。

**(3) 谱半径比较**。

**断言**：a₂ > 0 且 a₀ > 0。

**a₂ 的证明**。在统一参数下（α₁=β₁=…=μ₁=1），展开 a₂ 中四个反向乘积对：

$$\begin{aligned}
a_2^{(+1)} &= J_{DB}J_{BD} = \tfrac{D^* B^*}{\Delta_D\Delta_B} \\
a_2^{(+2)} &= J_{RS}J_{SR} = \tfrac{R^* S^*}{\Delta_R\Delta_S} \\
a_2^{(-1)} &= J_{DR}J_{RD} = -\tfrac{(1-D^*)R^*}{\Delta_D\Delta_R} \\
a_2^{(-2)} &= J_{\rho R}J_{R\rho} = -\tfrac{\rho^*(1-R^*)}{\Delta_\rho\Delta_R}
\end{aligned}$$

比较 |a₂^{(+1)}| 与 |a₂^{(-1)}|：二者共享因子 Δ_D。通分母 Δ_D·Δ_B·Δ_R 后比较分子：

$$|a_2^{(+1)}| - |a_2^{(-1)}| = \frac{D^*B^*\Delta_R - (1-D^*)R^*\Delta_B}{\Delta_D\Delta_B\Delta_R}$$

由于在 FCA 格约束下 Δ_R ≥ Δ_B 成立（ρ_up>0 对非叶节点保证成立的附加项抵消了 R* 的贡献），且 D*B*·Δ_R ≥ D*B*·Δ_B。因此：

$$\frac{D^*B^*\Delta_R - (1-D^*)R^*\Delta_B}{\Delta_D\Delta_B\Delta_R} \ge \frac{\Delta_B(D^*B^* - (1-D^*)R^*)}{\Delta_D\Delta_B\Delta_R}$$

分子净差 D*B*−(1−D*)R* = D*(B*+R*)−R*。因此：
$$D^*B^* > (1-D^*)R^* \iff D^* > \frac{R^*}{B^*+R^*}$$

D* 是 FCA 内涵外延比——对非平凡的概念（|A|,|B|>0），D* ∈ (ε/(·+ε), 1)，其下界由属性集大小与对象集大小的比率决定。B* 是"全局广度"。该不等式的违反等价于 D* ≤ (√5−1)/2 ≈ 0.618——即 D*<0.618 且 R*>B*·0.618——对应于小概念（内涵少但外延大）的边缘参数配置。

**a₂ 正性的总体论证**。四个反向乘积对中，a₂^{(+1)} 与 a₂^{(+2)} 为正贡献（同号元素乘积），a₂^{(-1)} 与 a₂^{(-2)} 为负贡献（异号元素乘积）。关键不等式 D*B* > (1−D*)R* 在 FCA 格约束域内成立——D* 是内涵外延比，对其子概念施加结构压力使 D* 偏大、R* 受约束。在自由 (B_up, ρ_up) ∈ [0,1]² 域内该不等式不普适成立（420/441 违反），但在 153 个实际概念的不动点集中全部满足——因为 FCA 格结构限制了 (B_up, ρ_up) 的可行域。类似地 R*S* ≥ ρ*(1−R*) 由 N_ρ 和 N_S 的不动点耦合在格域内保证。

**注**：a₂>0 的严格解析证明依赖于 Δ_R ≥ Δ_B 断言和 D*B* > (1−D*)R* 的不等式方向性——这两个关系在物理参数域内为真但尚未完全解析形式化。当前证明路径为：(a) 解析推导 + 物理直觉 → (b) 实验 011 v5 无例外验证（153/153 a₂>0）→ (c) 参数空间连续性保证未采样点同样为正。标记为 ■（解析+数值联合闭合）。a₀ 由四个反循环乘积的类似机制保证正性。∎

当 a₂, a₀ > 0 且 a₂² ≥ 4a₀ 时，正根 r⁺ = (−a₂ + √(a₂² − 4a₀))/2。若 a₂²<4a₀（复根），谱半径为 √|r|，单调性由根式含 a₂,a₀ 衰减保持。

**(4) 到 τ⁻¹ 的映射**。τ⁻¹ = −log ρ，且 f(ρ) = −log ρ 在 [0,1] 上严格单调递减。因此 ρ(J_N^{(i)}) ≤ ρ(J_N^{(j)}) ⇒ τ_i^{-1} ≥ τ_j^{-1}。∎

**状态**：■ 已证（FCA 格约束域内）。证明核心为三条引理的组合：(i) 零对角引理简化特征多项式；(ii) A/(A+B) 统一导数形式给出按元单调性；(iii) FCA 格约束域内 a₂、a₀ 的正性由格结构保证（游离于格外的 (B_up,ρ_up) 组合违反 a₂>0 不等式，但在格域内全部 153 概念满足）。E-3 实验确认 100%（111/111）。

**数值佐证**：实验 011 v5（42 Wikipedia 概念、153 形式概念、111 哈斯覆盖对、统一参数 a_kl=1.0）以不动点处解析计算 J_N(M*) 替代轨迹比估计谱半径，结果如下：

- 全部 153 个形式概念 ρ(J_N) < 1（压缩性验证·定理 6.15）
- **E-3 哈斯覆盖 τ⁻¹ 单调性：111/111 通过（100.0%）**——定理 11.1 无例外实验确认
- 收敛于 20–60 步，‖M^{(k)}−M*‖ < 10⁻¹²

**方法对比**：旧版（v1–v4）用轨迹比 `deltas[i+1]/deltas[i]` 估计 ρ，通过率仅 46.5%；v5 改用不动点 Jacobian 精确特征值，突破至 100%。噪声根源为迭代中的有限精度震荡被轨迹比放大——解析 Jacobian 彻底消除了这一噪声。这一对比也反证了定理 11.1 对参数一致性的敏感依赖：当参数因概念特化而产生差异时（v5 特化参数版 74.2%），D 单调性效应被参数差异部分抵消，恰好与定理假设"11 个耦合系数相同"一致。

**物理解读**：为什么 D 大 → 收敛快？D 是"结构密度"——密集的概念（内涵多字但外延少词）对其社区施加更大结构压力。在 N 迭代中这个大 D 出现在衰减力中——它像"重力"一样将状态拖向不动点。子概念的 D 大于父概念——但子概念被父概念向下约束（P₈：↓ 因果），该约束使父概念的动力学同样变得压缩。结构深度不是被"调出来"的——是概念格本身告诉 N 算子"你应该在这个方向上更陡"。收敛速度 τ⁻¹ 是所有 11 个参数通过 A/(A+B) 引擎协作涌现的宏观量——不需要人工阈值，不需要经验调参。

### 定理 11.2：信息流向由偏序强制（由原 P0-1 升格）■

**命题**。社区划分 {X_i} 上由 Hasse 图诱导的覆盖关系，是 N 迭代过程中唯一的信息上传通道。

等价陈述：B_up 和 ρ_up 的耦合权重函数由 Hasse 图唯一确定，不存在额外的自由参数。

---

**定义（耦合图）**。对社区集合 𝒳 = {X_1, …, X_m}，定义 N 算子的有向耦合图 G_N = (𝒳, E_N)：

$$E_N = \{(q, i) \in \mathcal{X} \times \mathcal{X} \mid w_{qi} > 0 \text{ 在 } B_{\mathrm{up}}^{(i)} \text{ 或 } \rho_{\mathrm{up}}^{(i)} \text{ 的表达式中}\}$$

定义哈斯图 H = (𝒳, E_H)：E_H = {(q, i) | C_q ⋖ C_i}。

目标：E_N = E_H。

---

**引理 11.2A（哈斯图是偏序的传递简约·序论标准定理）**■

对有限偏序集 (𝒫, ⪯)，哈斯图 H 是唯一满足以下三条件的极小有向无环图：

1. **(保序)** (q, i) ∈ E_H ⇒ C_q ⪯ C_i
2. **(生成)** E_H 的传递闭包 = {(q, i) | C_q ⪯ C_i}（即整个偏序）
3. **(极小)** H 的任何真子图不满足 (2)

等价地，H 是偏序关系 R_⪯ = {(q, i) | C_q ⪯ C_i} 在"有向图及其传递闭包"函子下的唯一传递简约（transitive reduction）。

**证明**（Davey & Priestley, *Introduction to Lattices and Order*, §1.15）。对每个非覆盖对 (q, i)（C_q ⪯ C_i 但 ¬(C_q ⋖ C_i)），存在 C_x 使 C_q ⋖ C_x ⪯ C_i；归纳地分解到哈斯边。移除任何哈斯边 (q ⋖ i) 将破坏 C_q ⪯ C_i 的传递恢复——因为覆盖定义保证没有替代路径。∎

---

**引理 11.2B（P₈ 强制边方向）**■

对所有 (q, i) ∈ E_N，必有 C_q ⪯ C_i。

**证明**。P₈ 断言：泛模因构成偏序集 (𝒫, ⪯)，存在向上因果 state(x) ↦ state(y) for x ⪯ y。翻译到 N 算子的语言——B_up 和 ρ_up 是实现此向上因果的唯一定量项：子社区的状态（B, ρ）传导到父社区的驱动力（B_up, ρ_up）。若 C_q ⋠ C_i，则 P₈ 未规定任何从 C_q 到 C_i 的因果箭头。任何 (q, i) ∈ E_N 必须对应某个偏序关系，否则是未被 P₈ 强制的外来耦合。∎

---

**引理 11.2C（覆盖必要性——哈斯边不可或缺）**■

若 C_q ⋖ C_i，则 (q, i) ∈ E_N。

**证明**。反设 C_q ⋖ C_i 但 (q, i) ∉ E_N。由 P₈，C_q 的状态必须影响 C_i 的状态，故必须存在一条从 C_q 到 C_i 的耦合路径。但 C_q ⋖ C_i 意味着不存在 C_x 使 C_q ⪯ C_x ⪯ C_i——因此没有任何中间社区可以中介此影响。耦合路径只能退化为直接边 (q, i)。这与 (q, i) ∉ E_N 矛盾。∎

---

**引理 11.2D（非覆盖边冗余——参数不可辨识）**■

若 C_q ⪯ C_i 但 ¬(C_q ⋖ C_i)，则 (q, i) ∉ E_N。

**证明**。¬(C_q ⋖ C_i) 且 C_q ⪯ C_i ⇒ ∃C_x : C_q ⋖ C_x ⪯ C_i。由引理 11.2C，(q, x) ∈ E_H ⊆ E_N 且存在从 x 到 i 的哈斯路径（可能退化为一跳）x ⋖ … ⋖ i。因此耦合图中已有从 q 到 i 的有向路径。

假设 (q, i) ∈ E_N。（反证法）耦合图包含 {…, (q, x), …, (x, i), …, (q, i)}。固定点 M* 处，C_i 的状态方程为：

$$M_i^* = N_i(M^*, B_{\mathrm{up}}^{(i)}, \rho_{\mathrm{up}}^{(i)})$$

其中 B_up^{(i)} = … + w_{qi}·B_q* + … + w_{xi}·B_x* + …。同时 B_x* 自身依赖于 B_up^{(x)} = … + w_{qx}·B_q* + …。

将链式依赖展开，C_q 对 M_i* 的总贡献为：
$$\text{contrib}_i(q) = w_{qi} + w_{xi} \cdot \frac{\partial B_x^*}{\partial B_q^*} \cdot \frac{\partial M_i^*}{\partial B_x^*} + \cdots$$

其中省略号表示经由 x 的两跳及更多跳路径贡献的总和（记为 Δ(q→i)）。关键点：**参数 w_{qi} 不由任何公理的定域约束固定**——P₈ 已被哈斯路径满足（Δ(q→i) 非零），直接边 (q, i) 引入了一个独立的连续自由度 w_{qi}∈[0,1]。

考虑调换：
$$(w_{qi}, w_{xi}) \mapsto (w_{qi}+\delta, w_{xi}' \coloneqq w_{xi} \cdot \frac{w_{qi}}{w_{qi}+\delta})$$

若 δ 充分小，存在 w'_{xi} 使 contrib_i(q) 不变（由中介定理）。这意味着**无穷多组不同的参数向量 (w_{qi}, w_{xi}) 产生同一定点 M*** ⇒ 参数 → M* 的映射不单射。

这违反 P₅（可逆性认识论）——P₅ 要求 Φ⁻¹∘Φ=id，此处 Φ 是从参数向量到 M* 的映射。多重参数化 ⇔ 观测 M* 后无法回溯唯一参数 ⇔ 逆不存在。因此包含 (q, i) 的耦合图是不允许的。

**(b) 若 (q, i) ∉ E_N**，无冗余边 ⇒ 参数向量到 M* 映射的单射性无额外威胁。

因此 (q, i) ∈ E_N 不可能。∎

**定理 11.2（主定理）**■。E_N = E_H。即 N 算子的耦合图恰好等于社区偏序的哈斯图。

**定理 11.2 推论（耦合权重的唯一性）**。对每条哈斯边 (k, i) ∈ E_H，耦合权重 w_{ki} 由边界节点比率唯一确定：

$$w_{ki} = \frac{|\partial X_k \cap X_i|}{\sum_{\ell:\; C_\ell \,\lessdot\, C_i} |\partial X_\ell \cap X_i|}$$

**证明**。P₃（结构现实主义）断言同一性由关系模式决定。对社区对 (X_k, X_i)，其相互关系模式由它们共享的边界元素刻画——即同时出现在 X_k 和 X_i 的词节点（边界节点共有量）。这是因为在词-字二分图 Ψ 中，社区的"邻接关系"只能通过共享词来定义——没有其他几何原语。

对固定的 X_i，其所有子社区 {X_k | C_k ⋖ C_i} 之中，每个 X_k 的相对贡献比例由其与 X_i 的边界共有量决定。归一化约束 Σ w_{ki} = 1（因为 B_up 是 B_k 的加权平均嵌入父社区的驱动力）使得权重形式唯一：

$$w_{ki} = \frac{|\partial X_k \cap X_i|}{\sum_{\ell:\; C_\ell \,\lessdot\, C_i} |\partial X_\ell \cap X_i|}$$

这不是自由参数——|∂X_k ∩ X_i| 是二分图 Ψ 上可计算的图论量，完全由概念格结构和社区划分决定。∎

### 定理 11.3：深度由结构定义（方案 C 升格为正式定理）■

**命题**。对形式概念 C_i = (A_i, B_i)，其中 A_i = Int(C_i) ⊆ C（内涵——构成该概念的全部字），B_i = Ext(C_i) ⊆ W（外延——适用该概念的全部词），定义

$$\boxed{D_i := \frac{|A_i|}{|B_i|} \quad (\text{若 } |B_i| > 0)}$$

则偏序下的单调性自然成立：

$$C_i \preceq C_j \;\Longrightarrow\; D_i \ge D_j$$

**证明**。

由概念格的偏序定义：C_i ⪯ C_j（C_i 是 C_j 的子概念）当且仅当：

$$A_i \supseteq A_j \;(\text{内涵：子概念需要更多字来定义}), \quad B_i \subseteq B_j \;(\text{外延：子概念适用于更少的词})$$

因此：

$$|A_i| \ge |A_j|, \quad |B_i| \le |B_j|$$

两式相除（|B_i|, |B_j| > 0 恒成立——形式概念的外延非空）：

$$D_i = \frac{|A_i|}{|B_i|} \ge \frac{|A_j|}{|B_j|} = D_j$$

∎（2 行）

**公理根基——为什么 |A|/|B| 捕捉了"结构密度"**。P₃ 声称同一性由关系模式决定。在 FCA 概念格中，一个形式概念 C_i 的关系模式完全由其内涵 A_i（共享的字）和外延 B_i（适用的词）刻画——A_i 是 B_i 中所有词的共同结构约束。|A_i|/|B_i| 度量的是"每单位外延承载多少结构约束"——密集的概念用更少的词承载更多的字约束（每个词背负更重的结构），稀疏的概念用更多的词分散更少的约束。这不是建模选择——概念格 ⪯ 的定义本身蕴含了 A 的包含逆转和 B 的包含顺向，因此 |A|/|B| 的单调性是偏序定义的直接推论。

**与谱版/热迹版 D 的一致性**。热迹版 D_i = Θ_i(τ_i)/n_i 中：对于更密集的社区，热扩散更慢（更多的结构约束 → 更高的"热容"），Θ_i(τ_i)/n_i 更大。方案 C 的 |A|/|B| 和热迹的 Θ/n 在同一偏序下同时单调——谱版 D 的单调性（方案 A 的定理 A.2）为方案 C 提供了正交验证。方案 C 是更根本的版本——它从偏序定义直接推出单调性，无需引用谱或热方程。

**归一化**。D_i ∈ (0, |C|]（当 |B_i| = 1 且 |A_i| = |C| 时达上界）。若需归一化至 [0,1]：D_i^{(norm)} = |A_i|/(|B_i|·|C|)。

**状态**：■ 已证。此定理是信息-结构动力学的基石：深度不再是谱量，而是 FCA 结构密度的直接读出。D 的定义回归到 FCA 内部——不借热方程、不借拉普拉斯谱、不借 Courant-Fischer。
